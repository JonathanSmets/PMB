Les conditions de vente

Les conditions de vente se composent :

  • de conditions de paiement : vous pouvez choisir dans la liste déroulante un des modes de paiements définis via Administration > Acquisitions > Modes de paiement.

  • des remises par type de produit : vous pouvez créer une remise en cliquant sur Ajouter une remise.

Création de remise⚓

Pour créer une remise vous devez choisir un type de produit dans une liste défini via Administration > Acquisitions > Types de produits.

Puis vous devez définir le pourcentage de remise que votre fournisseur vous octroie pour un type de produit et Enregistrer.

Finaliser les conditions de ventes⚓

Cliquez sur le bouton Enregistrer.

Truc & astuce

Vous pouvez définir plusieurs remises par fournisseur en cliquant sur Ajouter une remise.

La colonne Historique des relances vous permet de consulter les relances éventuelles faites à un fournisseur.
