Livraisons

Le Bon de Livraison se créé automatiquement dès que vous déclarez une commande en réception.

Vous pouvez entrer le n° du bon de livraison de votre fournisseur, puis vous devez pointer la livraison pour déclarer le nombre d'ouvrages reçu.

Pour déclarer le nombre d'ouvrages reçu, vous pouvez le saisir directement dans la cellule ou, après avoir coché la case auto, douchettez les ISBN ou codes des ouvrages réceptionnés.

L'action de pointer dans le bon de livraison, vous permet de contrôler si la totalité de la commande a été livrée. Dans ce cas votre commande sera considérée comme soldée.

Remarque

Si vous ne cochez pas la case auto, le fait de saisir ou douchetter les codes vous permet de chercher l'ouvrage concerné afin de modifier manuellement la quantité reçue.

Depuis cette fenêtre, vous pouvez également annuler une réception. En sélectionnant votre document puis Supprimer les lignes cochées. Vous retrouverez alors votre document dans le module du menu de gauche Réception.
