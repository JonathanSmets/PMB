Les suggestions multiples

Afin d'éviter une manipulation répétitive dans l'émission des suggestions, un module de suggestions multiples existe dans PMB, permettant l'émission de plusieurs suggestions en une seule fois.

Le formulaire de suggestions multiples est accessible uniquement sous Acquisitions > Suggestions multiples.

Dans un souci de visualisation globale, le formulaire de saisie des suggestions se présente linéairement.

Chaque ligne dispose des mêmes champs de saisie que le formulaire d'une suggestion unique :

  • le titre de l'ouvrage

  • l'auteur

  • l'éditeur

  • l'ISBN

  • le prix indicatif

  • l'URL associé (site internet)

  • un commentaire

  • un commentaire dédié à la gestion

  • la date de parution

  • une source

  • la quantité

Attention

Le formulaire de suggestions d'acquisitions multiples n'est pas actif immédiatement. La ligne présente à l'affichage a pour seule fonction l'ajout d'une ligne de saisie.

Par conséquent il est impératif de cliquer sur le bouton + en bout de ligne.

Définir une origine (si l'origine de la suggestion n'est pas la personne qui la saisit dans l'interface de gestion)⚓

Par défaut l'origine indiquée sera celle du compte utilisateur qui saisit la suggestion. Il est possible de la changer (dans le cas où on a fourni au professionnel une liste de suggestions provenant d'une tierce personne) en cliquant sur ..., il est possible de rechercher un autre compte utilisateur ou lecteur (emprunteur) dans la fenêtre popup. Avec la même manipulation, en Modification de suggestion, il est possible de rajouter d'autres lecteurs ou utilisateurs comme Autres demandeurs. Les lecteurs ajoutés depuis cette fonctionnalité peuvent suivre l'évolution de la suggestion depuis leur compte lecteur.

Par ailleurs, la suggestion peut s'appliquer à une seule localisation ou à tous les sites si la localisation des suggestions est activée.
