Tableau des suggestions⚓

En allant dans le menu de traitement des suggestions, vous trouverez un tableau permettant de rechercher parmi elles, et d'agir dessus :

  • en recherchant dans leurs champs (1)

  • en filtrant selon leur état, source ou origine (personne) (2)

  • en cherchant sur une période (3)

Une fois ces filtres appliqués, ne pas oublier de cliquer sur le bouton Rechercher

Depuis cet écran, il est également possible de faire une Création de suggestion (4)

Si une liste est affichée, il est possible de l'Imprimer (5), de les Exporter vers un format bibliographique (6) ou de faire un Export tableau (7)

Les différentes parties suivantes présentes la suite des statuts possibles pour une suggestion. Pour les modifier, il faut les cocher et ensuite cliquer sur le statut à leur appliquer (8).

Les suggestions à valider⚓

Une suggestion à valider est une suggestion émise et non traitée.

Vous pouvez :

  • Fusionner plusieurs suggestions.

  • Valider la suggestion, s'il s'agit d'une suggestion permettant d'identifier un ouvrage, validez la suggestion. Pour l'instant, il ne s'agit pas de savoir si l'ouvrage est disponible ou non, mais s'il est intéressant dans votre fonds.

  • Rejeter la suggestion : Les éléments fournis ne permettent pas d'identifier un ouvrage ; la suggestion est un message à l'attention du bibliothécaire/documentaliste, il s'agit d'une erreur, d'un test, d'un usage inapproprié, ... dans ce cas, la suggestion doit être rejetée.

Les suggestions validées⚓

Une suggestion validée correspond à une suggestion réceptionnée en attente de confirmation.

En plus de pouvoir la fusionner avec une autre suggestion, ou la supprimer, vous pouvez :

  • Confirmer la suggestion. Oui, la suggestion est bien un ouvrage identifié ; oui, cette suggestion correspond aux orientations de votre fonds. Confirmer la suggestion vous permet d'annoncer au lecteur que l'acquisition est bien envisagée.

  • Abandonner la suggestion. Oui, la suggestion est valide ; non, cette suggestion ne correspond pas aux orientations de votre fonds. Abandonner la suggestion signifie que son traitement n'ira pas plus loin.

Les suggestions confirmées⚓

Une suggestion confirmée correspond à un ouvrage entré dans le processus d'acquisition.

Vous pouvez :

  • Deviser l'ouvrage suggéré, signifie qu'une demande de devis va être effectuée.

  • Commander l'ouvrage suggéré.

  • Abandonner la suggestion.

Attention

Selon le paramétrage dans Administration > Outils > Paramètres > Acquisitions, un clic sur Commander lance soit la création d'un bon de commande (choisissez un fournisseur, les suggestions sont ensuite automatiquement insérées comme autant de lignes de la commande) soit vous permet de tenir informé votre lecteur du fait que vous lancez la commande car vous gérez probablement vos bons de commande par un autre moyen.

Les suggestions transmises pour une demande de devis⚓

Une suggestion devisée correspond à un ouvrage pour lequel un devis a été demandé.

Vous pouvez :

  • Commander l'ouvrage suggéré.

  • Abandonner le processus d'acquisition.

Les suggestions passées en commande⚓

Une suggestion commandée correspond à un ouvrage en attente de livraison.

vous pouvez à ce stade :

  • Recevoir l'ouvrage.

  • Abandonner la commande.

Les suggestions réceptionnées⚓

Vous pouvez Archiver la suggestion une fois que l'ouvrage est reçu afin d'en conserver la trace pour solder tout le processus de traitement de la suggestion.

L'archivage des suggestions⚓

Une suggestion archivée correspond à une suggestion dont le traitement est terminé. La provenance d'une suggestion archivée peut être diverse :

  • suggestion rejetée

  • suggestion abandonnée

  • suggestion reçue

Cet état vous permet de conserver une trace des traitements ou une suppression.

Être alerté de l’évolution d'une suggestion⚓

Le lecteur à l'origine ou lié à une suggestion peut suivre l'évolution de cette dernière depuis son compte-lecteur.

Si le paramétrage l'autorise, dans Administration > Outil > Paramètres le lecteur peut recevoir un e-mail à la réception de la suggestion.
