Sources des suggestions

Les sources de suggestions permettent de définir la provenance des suggestions.

Cette provenance sert à distinguer les suggestions d'achat pour les gérer par sources, et identifier le format à appliquer lors de l'import de suggestions depuis un fichier.

 

Pour plus d'informations, voir le chapitre : Les suggestions

L'ajout d'une source de suggestions se fait en cliquant sur le bouton Ajouter une source de suggestions, la modification en cliquant sur le libellé de la source de suggestions à modifier.

Chaque source de suggestions est définie par un unique libellé.
