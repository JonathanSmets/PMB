Gestion du calendrier

Attention

Par défaut, le calendrier présente l'année courante, pour la localisation définie dans les préférences de l'utilisateur Site de gestion par défaut des lecteurs.

Si plusieurs localisations sont renseignées dans le logiciel PMB, il est nécessaire de définir les jours d'ouverture et de fermeture pour chaque localisation, avec le sélecteur.

Définir une période d'ouverture⚓

  • Renseigner la période avec des dates de début et de fin au format JJ/MM/AAAA (en langue française).

  • Cocher les jours concernés

  • Ajouter éventuellement un commentaire.

  • Dupliquer pour : si vous avez d'autres localisations qui ont les mêmes dates d'ouverture, vous pouvez dupliquer ce calendrier.

  • Cliquer sur Ouvrir pour déclarer ces jours comme étant des jours d'ouverture (pour la localisation courante).

 

Une fois l'action réalisée, les dates d'ouverture apparaissent sur fond vert :

 

Les flèches et permettent de changer d'année.

ExempleExemple lié au calendrier précédent⚓

Un lecteur emprunte un document dont la date de retour du document tombe au 14 février. Étant donné qu'à cette date, la bibliothèque est fermée, la date de retour est alors automatiquement fixée au 16 février (premier jour ouvert suivant).

Définir une période de fermeture⚓

Après avoir indiqué les jours d'ouverture de la structure, il est possible de définir des jours de fermeture (vacances,jours fériés,...), deux solutions sont alors possibles :

  • pour une fermeture sur une période donnée, il faut renseigner la période de fermeture, cocher les jours de fermeture et cliquer sur Fermer.

  • pour une fermeture exceptionnelle il suffit de modifier manuellement les dates en cliquant sur les jours concernés.

S'il est possible d'associer un commentaire à chaque période, il est également possible d'ajouter un commentaire sur un jour précis.

Pour cela il faut cliquer sur le nom du mois correspondant et renseigner le commentaire avant d'Enregistrer.
