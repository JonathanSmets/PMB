Connecteurs entrants

Les connecteurs entrants implémentent les protocoles nécessaires à une interrogation depuis PMB de données issues de systèmes tiers.

L'accès aux propriétés d'un connecteur se fait en cliquant sur le nom du connecteur, l'affichage de la liste des sources en cliquant sur l'icône  .

L'ajout d'une source à un connecteur est possible en cliquant sur le bouton Ajouter une source, la modification en cliquant sur le nom de la source.

PMB propose à ce jour les connecteurs suivants :

  • Interrogation Z 39.50

  • Interrogation d'un serveur SRU

  • Moissonnage d'un entrepôt OAI

  • Connexion à un serveur Aligastore (Inactif)

  • Connecteur des bases NCBI Entrez

  • Import de fichiers

  • AWS ECommerce Service (Amazon)

  • Lehmanns
