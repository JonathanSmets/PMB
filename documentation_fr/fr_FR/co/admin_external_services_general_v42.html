Autorisations générales

Les autorisations générales ont été simplifiées dans la version 3.4 du logiciel.

Elles rappellent simplement quels sont les utilisateurs autorisés pour un groupe de fonctions ou une fonction (méthode).

Les utilisateurs autorisés sont précisés à droite de chaque groupe de fonctions ou fonction.

Il est possible d'accéder directement aux modifications des autorisations par utilisateur en cliquant sur le login concerné.
