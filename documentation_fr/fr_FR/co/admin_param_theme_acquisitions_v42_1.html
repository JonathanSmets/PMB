Activer les acquisitions⚓

Si la gestion des acquisitions n'est pas activée, vous ne verrez ni l'onglet Acquisitions, ni le menu Acquisitions dans l'onglet Administration.

Onglet Administration, menu Outils, sous-menu Paramètres, zone Acquisitions  :

Activation des acquisitions

Paramètre

Valeur par défaut

Explication

active

0

Module acquisitions activé.

0 : Non.

1 :Oui.

Attention

Cette activation est globale.

Il faut également autoriser l'accès à l'onglet Acquisitions pour chaque utilisateur concerné.

Pour plus d'informations, voir le chapitre : Gestion des utilisateurs

Configurer les acquisitions⚓

Onglet Administration, menu Outils, sous-menu Paramètres, zone Acquisitions  :

Configuration des acquisitions

Paramètre

Valeur par défaut

Explication

budget

0

Utilisation d'un budget pour les commandes.

0 :optionnel

1 : obligatoire

budget_show_all

0

Sélection d'une rubrique budgétaire en commande : toutes les afficher ? 0: Non (par pagination) 1: Oui.

custom_calc_numero

Fonction personnalisée de numérotation des actes d'achats.

 

Indiquer le nom du fichier et le placer dans le répertoire ...pmb/acquisition/achats/

email_sugg

0

Information par email de l'évolution des suggestions.

0 :Non

1 :Oui

 

Pour informer les personnes ayant effectué des suggestions.

format

8,CA,DD,BL,FA

Taille du Numéro et Préfixes des actes d'achats.

ex : 8,CA,DD,BL,FA

8 = Préfixe + 8 Chiffres

CA=Commande Achat, DD=Demande de Devis,BL=Bon de Livraison, FA=Facture Achat

gestion_tva

0

Gestion de la TVA.

0 :Non.

1 :Oui, avec saisie des prix HT.

2 :Oui, avec saisie des prix TTC.

poids_sugg

U=1.00,E=0.70,V=0.00

Pondération des suggestions par défaut en pourcentage.

U=Utilisateurs, E=Emprunteurs, V=Visiteurs.

ex : U=1.00,E=0.70,V=0.00

show_abt_in_cmde

sugg_categ

0

Affectation des suggestions à une catégorie de suggestions.

0 :Non

1 :Oui

sugg_categ_default

1

Identifiant de la catégorie de suggestions par défaut.

sugg_display

Nom de la fonction personnalisée d'affichage des suggestions.

 

Indiquer le nom du fichier et le placer dans le répertoire ...pmb/acquisition/suggestions/

sugg_localises

0

Activer la localisation des suggestions ?

0 : Pas de localisation possible.

1 : Localisation activée.

sugg_to_cde

0

Transfert des suggestions en commande.

0 : Non.

1 : Oui.

Activer les suggestions en OPAC⚓

Onglet Administration, menu Outils, sous-menu Paramètres, zone OPAC, sous-zone Modules affichés dans l'OPAC  :

Activation des suggestions en OPAC

Paramètre

Valeur par défaut

Explication

show_suggest

O

Proposer de faire des suggestions dans l'OPAC.

0 :Non.

1 : Oui, avec authentification.

2 : Oui, sans authentification.

Attention

Cette activation est globale.

Il faut également autoriser l'accès aux suggestions pour les lecteurs concernés.

Pour plus d'informations, voir le chapitre :

Statuts
