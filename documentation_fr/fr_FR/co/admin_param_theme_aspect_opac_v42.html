Paramétrer l'aspect de l'OPAC

Onglet Administration, menu Outils, sous-menu Paramètres, zone OPAC, sous-zone Options générales de fonctionnement de l'OPAC :

Paramètres d'aspect de l'OPAC

Paramètre

Valeur par défaut

Explication

authorized_styles

pmb4,genbib

Styles de l'OPAC autorisés, séparés par une virgule.

 

Les valeurs possibles sont les noms des sous-répertoires du répertoire ...pmb/opac_css/styles, sauf common

default_style

genbib

Style graphique de l'OPAC :

nomargin : sans affichage du bandeau de gauche

 

Les valeurs possibles sont les noms des sous-répertoires du répertoire ...pmb/opac_css/styles, sauf common

default_style_addon

Ajout de styles CSS aux feuilles déjà incluses ?

Ne mettre que le code CSS, exemple: body {background-color: #FF0000;}

default_lang

fr_FR

Langue par défaut de l'OPAC :

 

Valeurs possibles :

ar : Arabe,

br_FR : Breton (France)

ca_ES : Catalan (Espagne)

de_DE : Allemand

en_UK : Anglais

en_US : Américain

es_ES : Espagnol

fr_FR : Français

hu_HU : Hongrois

it_IT : Italien

ja_JP : Japonais

la_LA : Lao

nl_NL : Néerlandais

oc_FR : Occitan (France)

pt_BR : Portugais (Brésil)

pt_PT : Portugais (Portugal)

tr_TR : Turc

...

show_languages

1 fr_FR,it_IT,es_ES,ca_ES,en_UK,nl_NL,oc_FR

Afficher la liste déroulante de sélection de la langue ?

0 : Non.

1 : Oui.

 

Faire suivre d'un espace et des codes des langues possibles séparées par des virgules :

fr_FR,it_IT,en_UK,nl_NL,oc_FR...

Afficher l'onglet Votre compte dans la barre de navigation de l'Opac ?

0 : Non.

1 : Oui.

Paramètre

Valeur par défaut

Explication

aff_expl_localises

0

Activer l'affichage des exemplaires localisés par onglet ?

0 : désactivé.

1 : premier onglet affiche les exemplaires de la localisation du lecteur, le deuxieme affiche tous les exemplaires.

0

Activer les recherches similaires sur une notice :0 : Non1 : Activer la recherche "Dans le même rayon" et "Peut-être aimerez-vous"2 : Activer seulement la recherche "Dans le même rayon"3 : Activer seulement la recherche "Peut-être aimerez-vous"

book_pics_msg

Message sur le survol des vignettes des notices correspondant au chemin fourni par le paramètre book_pics_url

book_pics_url

URL des vignettes des notices, dans le chemin fourni, !!isbn!! sera remplacé par le code ISBN ou EAN de la notice purgé de tous les tirets ou points.

exemple : http://www.monsite/opac/images/vignettes/!!isbn!!.jpg

12

Nombre de bulletins affichés par page dans l'affichage d'un périodique.

collstate_order

0

H 1 : id d'un template de notice
