Paramètres généraux liés à la gestion des paniers

Paramètre

Valeur par défaut

Explication

confirm_delete_from_caddie

0

Action à réaliser lors de la suppression d'une notice située dans un panier :

0 : Interdire ;

1 : Supprimer sans confirmation.

2 : Demander une confirmation de suppression.

Paramètres liés à la recherche et au triâš“

Onglet Administration, menu Outils, sous-menu Paramètres, zone Paramètres généraux  :

Paramètres généraux liés à la recherche et au tri

Paramètre

Valeur par défaut

Explication

default_operator

0

Opérateur par défaut :

0 : OR.

1 : AND.

extended_search_auto

1

En recherche multi-critères, la sélection d'un champ ajoute celui-ci automatiquement sans avoir besoin de cliquer sur le bouton Ajouter ?

0 : Non.

1 : Oui.

multi_search_operator

or

Type d'opérateur de recherche pour les listes avec plusieurs valeurs:

or : pour le OU.

and : pour le ET.

nb_max_tri

50

Nombre maximum de notices pour lesquelles le tri est autorisé.

recherche_ajax_mode

1

Affichage accéléré des résultats de recherche: "réduit" uniquement, la suite est chargée lors du click sur le "+".

0 : Inactif.

1 : Actif.

title_ponderation

0.5

Majoration de la pondération des mots du titre.

Mettre 0 (zéro) pour interdire la majoration.

ATTENTION utiliser le point décimal

allow_term_troncat_search

0

Troncature à droite automatique :0 : Non 1 : Oui

search_cache_duration

0

Durée de validité (en secondes) du cache des recherches

search_exclude_fields

Identifiants des champs à exclure de la recherche tous les champs (liste dispo dans le fichier includes/indexation/champ_base.xml)

search_noise_limit_type

0

Ecrêter les résulats de recherche en fonction de la pertinence. 0 : Non 1 : Retirer du résultat tout ce qui est en dessous de la moyenne - l'écart-type2,ratio : Retirer du résultat tout ce qui est en dessous de la moyenne - un ratio de l'écart-type (ex: 2,1.96)3,ratio : Retirer du résultat tout ce qui est dessous d'un ratio de la pertinence max (ex: 3,0.25 élimine tout ce qui est inférieur à 25% de la plus forte pertinence)

search_relevant_with_frequency

0

Utiliser la fréquence d'apparition des mots dans les notices pour le calcul de la pertinence.0 : Non 1 : Oui

search_stemming_active

0

Activer le stemming dans la recherche0 : Désactiver1 : Activer

Paramètres liés aux noticesâš“

Onglet Administration, menu Outils, sous-menu Paramètres, zone Paramètres généraux  :

Paramètres généraux liés aux notices

Paramètre

Valeur par défaut

Explication

form_editables

0

Grilles de notices éditables ?

0 : Non.

1 : Oui.

keyword_sep

;

Séparateur des mots clés dans la partie indexation libre, espace ou ; ou , ou ...

latest_order

notice_id desc

Tri des dernières notices ?

notice_id desc : par id de notice décroissant: idéal mais peut être problématique après une migration ou un import.

create_date desc : par date de création.

nb_lastnotices

10

Nombre de dernières notices affichées en catalogue (dernières notices).

notice_controle_doublons

Contrôle sur les doublons en saisie de notice :

0 : Pas de contrôle sur les doublons.

1,tit1,tit2, ... : Recherche par méthode _exacte_ de doublons sur des champs, défini dans le fichier notice.xml

2,tit1,tit2, ... : Recherche par similitude

notice_fille_format

0

Affichage des notices filles :

0 : avec leurs détails (notice dépliable avec un plus).

1 : Juste l'entête.

notice_reduit_format

0

Format d'affichage des réduits de notices :

0 : titre+auteur principal

1 : titre+auteur principal+date édition

2 : titre+auteur principal+date édition + ISBN

3 : titre seul

P 1,2,3 : tit+aut+champs persos id 1 2 3

E 1,2,3 : tit+aut+édit+champs persos id 1 2 3

T : tit1+tit4

4 : titre+titre parallèle+auteur principal

H 1 : id d'un template de notice

pret_express_statut

2

Statut de notice à utiliser en création d'exemplaires en prêt express.

show_notice_id

0

Afficher l'identifiant de la notice dans le descriptif ?

0 : Non.

1 : Oui.

1,X : Oui avec préfixe X.

show_permalink

0

Afficher le lien permanent de l'OPAC en gestion ? 0 : Non. 1 : Oui.

type_audit

0

Gestion/affichage des dates de création/modification ?

0 : Rien.

1 : Création et dernière modification.

2 : Création et toutes les dates de modification.

use_uniform_title

0

Utiliser les titres uniformes ?

0 : Non.

1 : Oui.

Onglet Administration, menu Outils, sous-menu Paramètres, zone Paramètres généraux  :

Paramètres généraux liés aux exemplaires

Paramètre

Valeur par défaut

Explication

droits_explr_localises

0

expl_data

expl_cb,expl_cote,location_libelle,section_libelle,statut_libelle,tdoc_libelle

Colonne des exemplaires, dans l'ordre donné, séparé par des virgules : expl_cb,expl_cote,location_libelle,section_libelle,statut_libelle,tdoc_libelle #n : id des champs personnalisés expl_cb est obligatoire et sera ajouté si absent

expl_display_location_without_expl

0

Affichage de la liste des localisations sans exemplaire 0: Non 1: oui

expl_list_display_comments

0

Afficher les commentaires des exemplaires en liste d'exemplaires :

0 : non.

1 : commentaire bloquant.

2 : commentaire non bloquant.

3 : les deux commentaires.

expl_show_dates

0

Afficher les dates des exemplaires ?

0 : Aucune date.

1 : Date de création et modification.

2 : Date de dépôt et retour (BDP).

3 : Date de création, modification, dépôt et retour.

expl_show_lastempr

1

Afficher l'emprunteur précédent sur la fiche exemplaire ? 0 : Non. 1 : Oui.

expl_title_display_format

expl_location,expl_section,expl_cote,expl_cb

Format d'affichage du titre de l'exemplaire en recherche multi-critères d'exemplaires.

Les libellés des champs correspondent aux champs de la table exemplaires, ou aux id de champs personnalisés, séparés par une virgule.

Les champs disposant d'un libellé seront remplacés par le libellé correspondant à l'affichage.

Exemple: expl_location,expl_section,expl_cote,expl_cb

html_allow_expl_cote

0

Autoriser le code HTML dans les cotes exemplaires ? 0 : non 1 : oui

label_construct_script

Script de construction d'étiquette de cote.

 

Fichier placé dans le répertoire .../pmb/catalog/caddie

numero_exemplaire_auto

0

Autorise la numérotation automatique des exemplaires.

0 : non

1 : Oui, pour monographies et bulletins

2 : Oui, pour monographies seules

3 : Oui, pour bulletins seuls

numero_exemplaire_auto_script

gen_code/gen_code_exemplaire.php

Nom du fichier de script php pour la génération des codes d'exemplaires en automatique.

 

Fichier placé dans le répertoire .../pmb/includes/gen_code

prefill_cote

custom_cote_02.inc.php

Script personnalisé de construction de la cote de l'exemplaire.

 

Fichier placé dans le répertoire .../pmb/catalog/expl

prefill_cote_ajax

Script personnalisé de construction de la cote de l'exemplaire en ajax.

type_audit

0

Gestion/affichage des dates de création/modification ?

0 : Rien.

1 : Création et dernière modification.

2 : Création et toutes les dates de modification.
