Paramétrer les infopages

Paramétrage des infopages⚓

Onglet Administration, menu Outils, sous-menu Paramètres, zone OPAC, sous-zone Modules affichés dans l'OPAC :

Paramétrage des infopages

Paramètre

Valeur par défaut

Explication

show_infopages_id

Id des infopages à afficher au dessous de la recherche simple, séparées par des virgules.

show_infopages_id_top

Id des infopages à afficher au dessus de la recherche simple, séparées par des virgules.

show_onglet_help

0

Afficher l'onglet HELP avec les onglets de recherche et y affecter un lien vers une infopage.

0 : Non.

## : id de l'infopage.

Paramètres liés aux infopages⚓

Pour associer une infopage à une localisation particulière, voir le chapitre : Localisations

 

Pour paramétrer un éditeur HTML, voir le chapitre : Paramétrer l'éditeur javascript HTML
