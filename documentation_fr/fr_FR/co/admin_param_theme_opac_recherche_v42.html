Les paramètres liés à la recherche en OPAC

Les options de recherches en OPACâš“

Onglet Administration, menu Outils, sous-menu Paramètres, zone OPAC, sous-zone Options pour les recherches dans l'OPAC :

Paramètres liés aux recherches en OPAC

Paramètre

Valeur par défaut

Explication

allow_extended_search

1

Autorisation ou non de la recherche avancée dans l'OPAC ?

0 : Non.

1 : Oui.

allow_external_search

0

Autorisation ou non de la recherche par connecteurs externes dans l'OPAC ?

0 : Non.

1 : Oui.

allow_personal_search

0

Activer l'affichage de l'onglet des recherches personalisées ?

0 : Non.

1 : Oui.

allow_tags_search

0

Recherche par tag (mots clés utilisateurs) ?

0 : Non.

1 : Oui.

allow_tags_search_min_occ

1

Nombre minimum d’occurrences d'un tag pour être affiché, 1 pour tous.

allow_term_search

1

Autorisation ou non de la recherche par termes dans l'OPAC ?

0 : Non.

1 : Oui.

allow_term_troncat_search

0

Troncature automatique à droite ?

0 : Non.

1 : Oui.

default_operator

0

Opérateur par défaut :

0 : OR.

1 : AND.

extended_search_auto

1

En recherche multi-critères, la sélection d'un champ ajoute celui-ci automatiquement sans avoir besoin de cliquer sur le bouton Ajouter ?

0 : Non.

1 : Oui.

indexation_docnum_allfields

0

Activer par défaut la recherche dans les documents numériques pour la recherche "Tous les champs" ?

0 : Non activée.

1 : Activée.

modules_search_abstract

1

Recherche simple dans le champ résumé :

0 : interdite.

1 : autorisée.

2 : autorisée et validée par défaut.

-1 : également interdite en recherche multi-critères.

modules_search_all

2

Recherche simple dans l'ensemble des champs :

0 : interdite.

1 : autorisée.

2 : autorisée et validée par défaut.

-1 : également interdite en recherche multi-critères.

modules_search_author

2

Recherche simple dans les auteurs:

0 : interdite.

1 : autorisée.

2 : autorisée et validée par défaut.

-1 : également interdite en recherche multi-critères.

modules_search_category

1

Recherche simple dans les catégories :

0 : interdite.

1 : autorisée.

2 : autorisée et validée par défaut.

-1 : également interdite en recherche multi-critères.

modules_search_collection

1

Recherche simple dans les collections :

0 : interdite.

1 : autorisée.

2 : autorisée et validée par défaut.

-1 : également interdite en recherche multi-critères.

modules_search_content

0

Recherche simple dans le s notes de contenu :

0 : interdite.

1 : autorisée.

2 : autorisée et validée par défaut.

-1 : également interdite en recherche multi-critères.

Paramètres actuellement inutilisé.

modules_search_docnum

0

Recherche simple dans les documents numériques:

0 : interdite.

1 : autorisée.

2 : autorisée et validée par défaut.

-1 : également interdite en recherche multi-critères.

modules_search_indexint

1

Recherche simple dans les indexations décimales :

0 :interdite.

1 : autorisée.

2 : autorisée et validée par défaut.

-1 : également interdite en recherche multi-critères.

modules_search_keywords

0

Recherche simple dans les indexations libres (mots-clés) :

0 : interdite.

1 : autorisée.

2 : autorisée et validée par défaut.

-1 : également interdite en recherche multi-critères.

modules_search_publisher

1

Recherche simple dans les éditeurs :

0 : interdite.

1 : autorisée.

2 : autorisée et validée par défaut.

-1 : également interdite en recherche multi-critères.

modules_search_subcollection

1

Recherche simple dans les sous-collections :

0 : interdite.

1 : autorisée.

2 : autorisée et validée par défaut.

-1 : également interdite en recherche multi-critères.

modules_search_title

2

Recherche simple dans les titres :

0 : interdite.

1 : autorisée.

2 : autorisée et validée par défaut.

-1 : également interdite en recherche multi-critères.

modules_search_titre_uniforme

0

Recherche simple dans les titres uniformes :

0 : interdite.

1 : autorisée.

2 : autorisée et validée par défaut.

-1 : également interdite en recherche multi-critères.

multi_search_operator

or

Type d'opérateur de recherche pour les listes avec plusieurs valeurs :

or : pour le OU

and : pour le ET

nb_max_criteres_tri

3

Nombre maximum de critères de tri à afficher.

nb_max_tri

50

Nombre maximum de notices pour lesquelles le tri est autorisé.

recherches_pliables

0

Les cases à cocher de la recherche simple sont-elles pliées ?

0 : Non.

1 : Oui et pliée par défaut.

2 : Oui et dépliée par défaut.

3 : Invisibles.

search_other_function

Fonction complémentaire pour les recherches en page d'accueil.

search_show_typedoc

1

Affichage de la restriction par type de document pour les recherches en page d'accueil.

term_search_height

350

Hauteur en pixels de la frame de recherche par termes (si pas précisé ou zéro : par défaut 200 pixels).

term_search_height_botton

120

Hauteur de la partie supérieure de la frame de recherche par termes (en pixels).

term_search_n_per_page

50

Nombre de termes affichés par page en recherche par terme.

title_ponderation

0.5

Majoration de la pondération des mots du titre mettre 0 (zéro) pour interdire la majoration.

ATTENTION utiliser le point décimal.

Truc & astuce

Nous vous recommandons de rendre invisible le paramètre recherches_pliables en recherche simple et de proposer une recherche prédéfinie à la place ou en complément de la recherche multi-critères.

L'affichage des recherches en OPACâš“

Onglet Administration, menu Outils, sous-menu Paramètres, zone OPAC, sous-zone Options d'affichage des résultats de recherche :

Paramètres liés à l'affichage des résultats de recherche

Paramètre

Valeur par défaut

Explication

congres_affichage_mode

0

Mode d'affichage des congrès :

0 : Comme pour les auteurs,

1 : Ajout d'un navigateur de congrès.

nb_aut_rec_per_page

15

Nombre de notices affichées pour une autorité donnée.

nb_result_first_page

10

Nombre de notices à afficher lors d'une recherche pour le critère "Tous les champs".

notice_groupe_fonction

Quel fichier/fonction inclure pour la présentation des résultats si toutes les notices d'une recherche sont parmi les types mentionnés.

exemple : a,b text;c,d music;k photo fera include(text.inc.php) et appel à la fonction text()

search_results_per_page

10

Nombre de résultats affichés sur les pages suivantes.

show_empty_items_block

1

Afficher le bloc exemplaires même si aucun exemplaire sur la notice ?

0 : Non.

1 : Oui.

show_results_first_page

0

Affichage de résultats sur la première page lors d'une recherche pour "Tous les champs" ?

0 : Non.

1 : Oui.

surlignage

0

Surligner les mots recherchés ?

0 : pas de surlignage.

1 : surlignage obligatoire.

2 : surlignage activable.

3 : surlignage désactivable.
