Circulation

Paramètre⚓

  • Nb de lignes par page :

    définit le nombre de lignes qui seront affichées dans les listes de transferts visibles dans l'onglet circulation.

  • Nombre de jours avant que l'alerte ne s'affiche en retour :

    définit un nombre de jours au delà duquel une alerte est affichée en zone d'alerte si le document n'est pas retourné dans sa localisation d'origine.

  • Autorise le traitement par lot en envoi :

    permet de valider l'envoi d'un lot d'exemplaires (non recommandé).

  • Autorise le traitement par lot en réception :

    permet de valider la réception d'un lot d'exemplaires (non recommandé).

  • Autorise le traitement par lot pour les retours :

    permet de valider le retour d'un lot d'exemplaires (non recommandé).

Attention

Les traitements par lot sont sources d'erreurs.

Prêt des exemplaires⚓

  • Action par défaut si le document est en transfert :

    définit si un document en transfert peut être prêté par défaut (Interdire le prêt est recommandé).

Retour des exemplaires⚓

  • Empêcher le retour de l'exemplaire sur un site autre que le sien :

    Ce paramètre bloque le retour d'un document si le document n'est pas rendu sur le bon site .

    Le message "Erreur : Cet exemplaire doit être rendu sur son site d'origine" est affiché.

  • Autoriser à forcer le retour de l'exemplaire sur un site autre que le sien :

    Si ce paramètre et le précédent sont simultanément activés, vous pourrez forcer le retour, de façon à outrepasser le blocage.

    Si le paramètre "Empêcher le retour..." ci-dessus n'est pas activé, ce paramètre sera sans effet.

  • Action par défaut lors d'un retour sur un autre site :

    permet de choisir si l'action par défaut lors du retour doit être :

    • A mettre de côté mis de côté pour être traité ultérieurement.
    • Changer localisation exemplaire.
    • Générer un transfert (recommandé).

  • Sauvegarder l'ancienne localisation en cas de changement de localisation au retour :

    permet choisir de conserver l'information de l'ancienne localisation (par défaut) ou non.

  • Etat du transfert généré :

    permet de préciser si l'on veut générer directement un envoi direct vers le site d'origine ou non (recommandé). Dans le deuxième cas, les exemplaires devront être lus à la douchette pour valider l'envoi.

  • Motif du transfert généré automatiquement :

    Si le transfert est automatiquement généré lors du retour de prêt, précise le motif de l'envoi.

  • Autorise une autre action que celle par défaut :

    permet de choisir une autre action lors du retour, que celle définie par défaut avec le paramètre Action par défaut lors d'un retour sur un autre site.

  • Générer un transfert pour répondre à une réservation :

    permet de choisir si l'on souhaite ou non générer une demande de transfert au retour d'un document réservé dans une autre localisation.

    Dans le cas où cette option est activée une alerte apparaît lors du retour informant qu'une demande de transfert est à traiter.
