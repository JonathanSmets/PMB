Général

Les paramètres généraux concernant les transferts sont des paramètres liés aux procédures communes à tous les transferts : étape de validation, nombre de jours de dépôts, statuts pendant et après les transferts.

Paramètreâš“

  • Activation de l'étape de validation :

    permet d'activer une étape de validation (acceptation préalable) pour toute demande de transfert.

  • Regrouper la gestion des départs : Validation, envoi, retour :

    Si cette fonctionnalité est activée le menu Départ dans l'Onglet Circulation permettra de valider les envois et les retours à partir du même écran (non conseillé dans un premier temps).

  • Nb de jours de dépôt par défaut :

    définit une durée par défaut pour les transferts.

  • Statut de l'exemplaire après validation :

    définit un statut pour les exemplaires dont le transfert a été validé mais dont l'expédition n'est pas encore réalisée.

  • Appliquer ce statut avant la validation :

    définit si le statut d'exemplaire après validation est également valable avant la validation (afin de bloquer le prêt d'un exemplaire en demande de transfert par exemple).

  • Statut de l'exemplaire pendant le transfert :

    définit un statut pour les exemplaires dont l'expédition est en cours.

Conseil

Il est préférable de définir des statuts d'exemplaires spécifiques concernant les transferts, ces statuts ne permettant ni le prêt, ni le transfert :

  • En attente de transfert : non prêtable, réservable, non transfèrable.

  • En transfert (signifie que le document est dans la navette) : non prêtable, réservable, transfèrable.

 

Pour plus d'informations, voir le chapitre : Statuts
