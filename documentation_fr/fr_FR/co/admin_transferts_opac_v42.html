Réservation

Ces paramètres permettent de régler le fonctionnement de la réservation de documents entre différents sites.

  • Choix pour la réservation :

    permet de définir dans quel site le lecteur doit retirer l'exemplaire réservé :

    • Localisation du lecteur (bibliothèque/ centre de documentation où il est inscrit).

    • Choix de la localisation (lors de la réservation).

    • Lieu fixe précisé (par exemple un lieu central où les accès sont facilités).

    • Localisation de l'exemplaire.

  • Site si lieu fixe précisé :

    Si le paramètre Choix pour la réservation, est défini sur "lieu fixe" , précise le lieu unique de retrait des exemplaires réservés.

  • Motif du transfert généré automatiquement :

    permet de préciser le motif enregistré lorsqu'un transfert est généré automatiquement pour une réservation.

  • Type de transfert généré :

    permet de définir si le transfert génère un envoi direct, ou doit donner lieu à une la validation (par lecture à la douchette du code-barres des exemplaires).
