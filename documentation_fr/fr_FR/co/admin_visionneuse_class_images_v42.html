Classe images

Cette classe prend en charge l'affichage des images de type :

  • PNG (mimetypes[1] 'image/png', 'image/x-png')

  • JPEG (mimetypes[1] 'image/jpg', 'image/jpeg')

  • GIF (mimetype[1] 'image/gif')

Au niveau du paramétrage, il est possible de spécifier :

  • La largeur et la hauteur maximum de l'image.

    (Une image plus petite n'est pas redimensionnée.)

  • Le nom d'une image Watermark (filigrane) ainsi que la transparence de celle-ci.

    L'image Watermark est à stocker dans le répertoire .../pmb/opac_css/images (ou l'un de ses sous répertoires).

En OPAC, si la recherche effectuée comporte des documents numériques (pris en charge par la visionneuse), ceux-ci sont affichables, soit en cliquant sur la vignette du document numérique correspondant, soit en cliquant sur le lien Visionner les documents numériques.

Le document sélectionné s'ouvre alors dans une fenêtre en avant plan (iframe[2]).

  • Le système de pagination permet de consulter les documents numériques du résultat de recherche, sans refermer la fenêtre d'aperçu.

  • Le lien Plein écran propose l'agrandissement de la fenêtre

  • Un clic sur le lien Fermer ou en dehors de la fenêtre referme celle-ci.

Références

  1. MIMEtype

    Multipurpose Internet Mail Extensions Type MIME Content-type

    Un MIMEtype est un système permettant d'identifier un format de données.

    Il est composé à minima d'un type et d'un sous-type (ex : application/pdf, image/gif).

  2. iframe

    Une iframe est une page HTML incorporée dans une autre page.
