Serveurs

Les serveurs Z39.50 définis ici sont accessibles pour la recherche Z39.50 dans l'onglet Catalogue.

L'ajout d'un serveur se fait en cliquant sur Ajouter, la modification en cliquant dans la liste sur le serveur à modifier.

Chaque serveur Z39.50 possède un profil reprenant l'ensemble des paramètres nécessaires à sa définition.

Pour obtenir le profil d'un serveur z39.50 , il est possible :

  • de contacter l'administrateur du serveur Z39.50.

  • d'effectuer une recherche portant sur les termes "profil Z39.50" ou "serveur Z39.50" dans un moteur de recherche.

  • d'utiliser des annuaires spécialisés tels que : http://irspy.indexdata.com/

Remarque

PMB est livré avec les paramètres de différents serveurs correspondant à des besoins francophones et à la bibliothèque du Congrès américain.

Le paramétrage d'un serveur Z39.50 est composé de différents éléments :

  • Nom  du serveur, qui sera utilisé à l'affichage.

  • Utilisation : CATALOG (catalogage). PMB permet uniquement l'utilisation du Z39.50 pour le catalogage.

  • Base : nom de la base de données (information contenue dans le profil Z39.50 du serveur).

  • URL et Numéro de port : adresse du serveur et numéro de port à interroger (information contenue dans le profil Z39.50 du serveur).

  • Format : Format des données (Information contenue dans le profil Z39.50 du serveur). Certains serveurs proposent plusieurs formats.

  • langue du format SUTRS : (Information contenue dans le profil Z39.50 du serveur).

  • Utilisateur et Mot de passe : identifiants de connexion au serveur (Information contenue dans le profil Z39.50 du serveur)

  • Fichier script de paramétrage de l'import Z39.50 : permet de préciser, en fonction du serveur, un script de traitement de l'import différent de celui défini par défaut.

Après avoir renseigné ces différents éléments, cliquez sur Enregistrer.

Attributs⚓

Le profil Z39.50 du serveur précise également différents attributs qui définissent les critères de recherche proposés par le serveur Z39.50.

Lors de la création du serveur, des attributs par défaut sont définis. Revenez en modification sur le serveur et cliquez sur Attributs pour les redéfinir.

L'ajout d'un attribut se fait en cliquant sur Ajouter, la modification en cliquant dans la liste sur l'attribut à modifier.

Chaque attribut se compose d'un Libellé à choisir dans une liste et d'un Attribut numérique.

Conseil

N'oubliez pas de vérifier que le protocole Z39.50, ainsi que les flux vers URL et numéros de port définis pour chaque serveur sont bien autorisés dans votre environnement réseau.
