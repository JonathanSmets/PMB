Personnalisation des formulaires de création des concepts

Pour pouvoir modifier les formulaires, dans Administration > Outils > Paramètres > Paramètres généraux, le paramètre form_editables doit valoir 1.

Attention

Pour personnaliser votre grille de catalogage, vous devez être connecté en tant que super utilisateur de PMB.

Il s'agit de l'utilisateur qui a l'identifiant "1" dans son url (id=1) et le login "admin" par défaut en Administration > Utilisateurs.

Pour personnaliser les formulaires des concepts, dans l'onglet Autorités > Concepts cliquez sur le bouton Ajouter un concept.

Cliquez ensuite sur le bouton Éditer format en haut à droite de l'écran.

Les champs sont alors encadrés ce qui signifie que vous pouvez opérer une action sur chacun d'entre eux en appuyant simultanément sur la touche Ctrl de votre claviez et en faisant un clic gauche dans le cadre concerné.

Un menu déroulant apparaît :

Pour le champ Dans les schémas :

  • Passer au premier plan : permet au champ de monter au premier niveau dans l'interface.

  • Monter : permet au champ de monter d'un cran dans le formulaire.

  • Descendre : permet de descendre le champ d'un cran dans le formulaire.

  • Passer en dernier : permet de au champ de descendre au dernier niveau dans l'interface.

  • Rendre invisible : permet de rendre invisible le champ dans le formulaire.

  • Déplacer dans la zone : permet de placer le champ dans une zone créée au préalable.

  • Sauver : permet de sauvegarder le formulaire modifié.

Zone :

Cette fonctionnalité permet d'éditer des zones au sein desquelles ils est possible de rassembler des champs. Pour cela il faut :

  • cliquer sur Créer une zone

  • la fenêtre suivant apparaît :

Il faut donner un nom à la zone, il est possible de faire afficher celle-ci sous forme d'onglet. Cliquez sur OK pour créer la zone. Ensuite à l'aide de la fonctionnalité Déplacer dans la zone vous pourrez associer des champs à celle-ci.

Zone par défaut :

Cette fonctionnalité permet d'opérer des modification sur une zone entière :

  • Monter la zone : permet à la zone de monter d'un cran dans le formulaire.

  • Descendre la zone : permet de descendre la zone d'un cran dans le formulaire.

  • Rendre invisible la zone : permet de rendre invisible la zone dans le formulaire.

  • Éditer la zone : permet de modifier les informations de la zone (nom etc.).

  • Rendre visible : ici sont listés les champs rendus invisibles, il est possible de les réactiver en cliquant sur le libellé.

Attention

Ne pas oublier de sauver les modifications faites via ce menu contextuel. Seul ce mode de sauvegarde permet de retrouver la mise en forme pour les concepts à venir.
