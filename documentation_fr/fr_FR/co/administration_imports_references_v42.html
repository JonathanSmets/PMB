Références pour l'import UNIMARC

L'import des notices est basé sur la norme UNIMARC[1], celui des exemplaires sur la Recommandation 995[2].

 

La norme UNIMARC[1] est une norme permettant de définir de nombreuses données.

Ces données peuvent être de portée limitée, sont parfois sujettes à des interprétations différentes ou encore spécifiques à une utilisation particulière.

 

Pour la réalisation de l'import, le logiciel PMB se base donc sur un ensemble de données pré-définies, dont le détail est décrit dans les fichiers suivants :

  • .../pmb/doc/import_unimarc.txt

  • .../pmb/doc/import_unimarc_lien.txt

 

Cet import peut être modifié par script de façon à se conformer à d'autres exigences.

Références

  1. UNIMARC

    Basé sur le format MARC (MAchine-Readable Cataloging), UNIMARC en est une variante créée par l'IFLA en 1977.

    Ce format est le format officiel d'échange de données bibliographiques en France.

  2. Recommandation 995

    La recommandation 995 a pour objectif de faciliter les échanges de données liées au prêt ou au dépôt d'exemplaires entre établissements gérés par des systèmes informatiques différents, notamment dans les organismes dépositaires de documents des bibliothèques départementales de prêt et dans les réseaux intercommunaux de bibliothèques.

    Elle porte sur la fourniture, au sein d'une notice bibliographique au format d'échange UNIMARC, de données relatives à l'exemplaire prêté ou échangé ainsi que d'autres informations permettant les classements, tris et statistiques.
