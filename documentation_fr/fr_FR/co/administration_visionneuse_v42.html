Visionneuse

Dans l'OPAC[1], le fait de cliquer sur une icône de document numérique provoque un envoi de ce document à destination du navigateur.

A réception du document, en fonction du mimetype[2] de celui-ci, le navigateur peut :

  • Ouvrir le document

  • Proposer son téléchargement, ou son affichage dans une application tierce.

Exemple

  • Une image au format .GIF sera par exemple ouverte dans le navigateur.

  • Un fichier au format .PDF pourra être affiché par le navigateur si celui-ci possède une extension (plugin) le permettant, proposé au téléchargement ou affiché dans une application telle qu'Acrobat Reader.

Remarque

En ce qui concerne les applications, le comportement peut être redéfini dans les options du navigateur.

La Visionneuse propose un mécanisme de redéfinition de l'affichage des documents numériques dans l'OPAC[1].

Le paramétrage définit les mimetypes[2] pour lesquels les documents sont ouverts dans la visionneuse. Cet affichage est réalisé dans la page (iframe[3]).

La visionneuse propose un ensemble de classes d'affichage qui prennent en charge certains mimetypes[2]. Pour les autres, la visionneuse laisse le navigateur gérer l'affichage.

Références

  1. OPAC

    Online Public Access Catalog. (Catalogue public en ligne)

  2. MIMEtype

    Multipurpose Internet Mail Extensions Type MIME Content-type

    Un MIMEtype est un système permettant d'identifier un format de données.

    Il est composé à minima d'un type et d'un sous-type (ex : application/pdf, image/gif).

  3. iframe

    Une iframe est une page HTML incorporée dans une autre page.
