Gérer plusieurs thésaurus

En fonction du paramétrage, il est possible de définir et gérer un thésaurus multilingue, voire plusieurs thésaurus

Pour plus d'informations, voir le Guide de l'administrateur PMB.

Paramétrer les thésaurus

L'ajout d'un thésaurus se fait en cliquant sur Modifier les thésaurus.

Cliquer sur Ajouter un thésaurus, la modification en cliquant sur le nom du thésaurus à modifier.

Le thésaurus nécessite :

  • Un Libellé,

  • Une Langue par défaut, qui correspond à la langue dans laquelle les catégories seront obligatoirement renseignées.

    Si d'autres langues sont prévues, la saisie des catégories dans ces langues sera facultative.

Attention

Le choix de la langue par défaut ne pourra pas être modifié après création.

En modification, est indiqué le numéro identifiant le thésaurus, utile pour le paramétrage.
