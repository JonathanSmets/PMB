Les concepts

ConceptVocabulaire SKOS⚓

SKOS ou Simple Knowledge Organization System est un standard permettant de représenter des thésaurus ou d'autres types de langages documentaires dans le cadre du Web Sémantique.

ComplémentLe Web Sémantique⚓

Le Web ou Toile Sémantique est un ensemble de standards et de technologies permettant de représenter tout type de modèle de données, aussi complexe soit-il. Il s'agit en outre de rendre ces données interrogeables dans un langage universel et interprétable par les machines. Tout cela en utilisant les technologies du web en vigueur.

En bref, le Web Sémantique fournit un modèle et des normes qui permettent à l'information d'être partagée et réutilisée entre différentes applications et groupes d'utilisateurs.

Concepts / Schémas / Collections / Collections ordonnées...⚓

... sont les composants du vocabulaire SKOS qui vont permettre de réaliser le portage des systèmes d'organisation de connaissance existant vers le Web Sémantique.

Concepts⚓

Si les thésaurus sont composés de termes (ou descripteurs), c'est-à-dire un terme choisi pour décrire un champ d'application, le SKOS utilise lui la notion de concept. Ce dernier traduit une idée ou un objet et peut-être associé à plusieurs termes qui le décrivent dans une langue. Quand dans un thésaurus on s'accorde sur l'utilisation d'un terme pour décrire une idée et l'on rejette ses synonymes, dans le SKOS le concept permet de délimiter une idée à laquelle on va associer tous les termes qui peuvent la nommer.

Remarque

La frontière entre un descripteur et un concept peut sembler ténue mais elle permet pourtant de cerner la démarche du SKOS. Dans sa manière d'englober plutôt que de rejeter des termes, le concept ajoute une dimension universelle à l'indexation. En effet, une idée peut être traduite par plusieurs termes, quelle que soit la langue dans laquelle celle-ci va s'exprimer. Cela rentre dans la logique du web sémantique et de la représentation universelle des modèles de données.

Schémas⚓

Un schéma est un ensemble de concepts rassemblés pour des raisons d'unité thématique ou de commodité d'utilisation. Il correspond plus ou moins à la notion de thésaurus à la différence qu'il ne s'agit pas ici d'une liste fermée. En effet, un concept peut être utilisé dans plusieurs schémas.

Schémas

Collections⚓

Une collection est un ensemble de concepts partageant une thématique commune. Une collection est l'équivalent d'un micro-thésaurus.

Collection

Collections ordonnées⚓

Une collection ordonnée est une collection au sein de laquelle les concepts doivent être placés dans un ordre précis.

Recherche⚓

L'écran de recherche des concepts est composé d'un formulaire de recherche et d'une liste de résultats.

Le formulaire de recherche comporte :

  • une zone de saisie des critères de recherche (dans laquelle les expressions booléennes sont acceptées)

  • un sélecteur pour filtrer la recherche sur un statut de concept

  • un sélecteur pour opérer la recherche sur un schéma de concept

  • un lien Voir les derniers concepts créés

  • une case à cocher pour faire une recherche uniquement sur les tops concepts, c'est-à-dire les concepts de premier niveau dans un schéma

Truc & astuce

Il s'agit ici d'une recherche plein texte sur le contenu de tous les champs associés au concept (libellés alternatifs, notes etc.).

La liste de résultats, paginée, précise également le nombre d'utilisations des concepts dans le catalogue.

Un clic sur le nombre d'utilisations renvoie la liste des notices pour lesquelles le concept correspondant est référencé.

  • le bouton TOP permet de remonter au premier niveau (racine) du schéma

  • les icônes indiquent qu'il existe des concepts enfants (spécifiques) et permettent de les afficher

  • les icônes  indiquent que l'on est sur une catégorie qui ne comporte pas de concept enfant (spécifique)

  • cliquer sur le nom d'un concept permet d'accéder directement au formulaire de modification

  • les icônes permettent de visualiser au sein d'une interface spécifique le concept, les informations et les liens le concernant ainsi que les notices dans lesquelles il est utilisé

  • un carré de couleur est parfois visible, il s'agit de la couleur associée au statut du concept

Création /modification⚓

La création d'un concept se fait avec le bouton Ajouter un concept, la modification en cliquant sur le libellé de celui-ci dans la liste.
