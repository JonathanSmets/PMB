Dictionnaire des mots vides

ConceptMot vide⚓

Un mot vide est un mot qui n'est pas porteur d"information. Il ne sera pas pris en compte lors d'une recherche.

Par défaut dans PMB, certains mots sont déjà déclarés vides en fonction de la langue. Pour le français ce sont : le, la, les, l', du, de, des, d', a, un, une, et, t, au, aux, en.

Vous pouvez étendre cette liste avec des mots vides qui seront définis pour toutes les langues.

Attention

Les mots vides concernent la recherche sur les titres.

Il existe trois types de mots que vous pouvez définir ici :

  • des mots vides calculés

    En définissant un pourcentage comme seuil de définition pour le calcul de mots vides, vous pouvez demander à PMB de chercher pour vous les mots vides dans votre base. 50% est la valeur par défaut.

    Si leur taux d'utilisation est supérieur à cette valeur, vous pouvez considérer que ces mots ne sont pas pertinents dans la recherche, trop utilisés ils ne permettent pas une recherche précise (par exemple, physique dans un laboratoire de physique théorique).

    Vous pouvez modifier ce pourcentage et cliquer sur Actualiser pour définir une liste de mots vides calculés.

  • des mots vides saisis

    Cliquez sur le bouton Ajouter un mot vide pour définir un Mot vide saisi. Celui-ci viendra s'ajouter à la liste des mots vides en bas de l'écran.

  • des mots non vides

    Si dans votre base, vous avez enregistré des documents concernant l'Association des Utilisateurs Xerox (AUX), un lecteur saisissant AUX ne pourrait obtenir de résultats sans définir une recherche exacte. Il est donc intéressant de définir le mot AUX comme étant non vide.

    Note : l'AUX existe vraiment.

Ajouter un mot vide⚓

Dans la page de gestion des mots vides, vous pouvez cliquer sur Ajouter un mot vide et remplir le formulaire.

Calculer un mot vide⚓

Pour calculer les mots vides, donnez un seuil en pourcentage et cliquer sur Actualiser.

ExempleCalcul avec un seuil à 70%⚓

Saisissez "70" dans la case de pourcentage et cliquez sur Actualiser.

Le calcul fait, PMB vous présente une liste.

Après un calcul :

  • Mot vide calculé : Cela veut dire que PMB considère que ce mot est un mot vide après calcul.

Si Il y a plus de 70 % des notices qui comportent un mot, le logiciel le calculera comme mot vide.

Cette liste peut changer après un nouveau calcul.
