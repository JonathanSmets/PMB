Afficher les avis et proposer le sien

Dans l'OPAC, les lecteurs peuvent :

  • consulter les avis déposés sur la notice d'un document

  • proposer leurs propres avis

ProcédureConsulter les avis sur une notice⚓

  1. Selon le paramétrage, les avis peuvent être visibles par tous ou uniquement après connexion au compte utilisateur.

  2. Accéder à la notice d'un document

    Le lecteur recherche le titre d'un document et affiche sa notice.

    Selon le paramétrage, l'avis peut apparaître :

    - vers le titre de la notice, dans ce cas, seule la note apparaît, il faut cliquer sur cette note pour avoir accès aux avis complets.

    - en dessous du bloc d'exemplaires, après avoir déplié la notice, dans ce cas, tout le commentaire apparaît.

    Remarque

    Le nombre d'étoiles correspond à la note moyenne qu'ont donné les lecteurs au document (sur une échelle de 0 à 5).

  3. Visualiser les avis

    Dans le cas ou la note apparaît vers le titre, le lecteur peut cliquer sur le nombre d'avis ou sur les étoiles pour ouvrir un pop-up dans lequel seront affichés les avis des lecteurs. Depuis cette interface, il peut consulter les avis existants ou, selon le paramétrage prévu, en ajouter un.

RemarqueAffichage des notices sans avis vers le titre⚓

Un message indique qu'aucun avis n'a été posé sur la notice :

RemarqueAffichage des notices sans avis après le bloc d'exemplaires.⚓

ProcédureDéposer un avis sur une notice⚓

  1. Accéder à la notice du document

    Selon le paramétrage, le lecteur doit être connecté, ou non, à son compte lecteur pour ajouter un avis.

    Le lecteur doit, dans un premier temps, rechercher à l'OPAC le document sur lequel il souhaite donner son avis.

  2. Afficher les avis et accéder au formulaire de saisie

    Le lecteur clique sur les avis déjà déposés par d'autres lecteurs, ou bien sur le message "Aucun avis sur cette notice".

    Dans le pop-up qui s'affiche, il clique ensuite sur le lien "Ajoutez votre avis".

    Dans le pop-up, un formulaire de saisie de l'avis s'affiche alors.

  3. Donner son avis

    Le lecteur donne son avis sur le document en remplissant le formulaire.

    Le lecteur valide son avis en cliquant sur Envoyer.

    L'avis n'est pas visible immédiatement. Il le sera après validation de la part des utilisateurs de PMB.
