Stockage dans l'entrepôt d'archivage

Les éléments supprimés manuellement sont automatiquement stockés dans l'entrepôt d'archivage lorsque celui-ci est activé.

Le stockage est également proposé lors de la suppression à partir d'un panier de notices ou d'exemplaires.

Sélectionnez l'entrepôt dans lequel stocker les éléments que vous allez supprimer avant de cliquer sur le bouton Supprimer.

Attention

Les notices d'un panier ne peuvent être supprimées (et donc archivées) si elles possèdent des exemplaires.

De même, Les exemplaires d'un panier ne peuvent être supprimés (et donc archivés) s'ils sont en cours de prêt.

Remarque

Lors de l'archivage, les exemplaires d'une même notice seront stockés unitairement (chacun avec un copie de la notice d'origine).
