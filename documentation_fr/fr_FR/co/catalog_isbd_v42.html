Affichage d'une notice

Sommaire
  1. La visualisation
  2. Les actions
  3. Les informations complémentaires sous la notice

La visualisation⚓

Après enregistrement, suite à un import, en cliquant sur un titre dans un résultat de recherche, vous visualisez la notice :

Le titre propre et l'auteur principal apparaissent en évidence pour identifier la notice. Il est également possible de paramétrer l'affichage de l'année de publication (optionnel).

En fonction des informations remplies dans le formulaire, le pavé ISBD est présenté ainsi que l'image de couverture.

ConceptISBD⚓

L'International Standard Bibliographic Description ou ISBD (description bibliographique internationale normalisée) désigne un ensemble de normes internationales de description de catalogage. Ces normes sont définies par l'IFLA, Fédération internationale des associations de bibliothécaires et d'institutions. (source : Wikipédia)

Dans l'interface de gestion de PMB, les notices sont affichées dans un format ISBD simplifié.

Remarque

La visualisation automatique via l'ISBN des images de couverture nécessite un paramétrage dans l'onglet Administration.

Pour plus d'informations, voir le Guide de l'administrateur PMB.

Paramétrer l'affichage des vignettes

Les actions⚓

Modifier⚓

Le bouton Modifier permet de retourner sur le formulaire de la notice. Vous pouvez ainsi modifier, ajouter ou supprimer des informations dans la notice.

Remplacer⚓

Les boutons Remplacer permettent le dédoublonnage de notices. La notice remplacée n'existe plus. Ses exemplaires sont rattachés à la notice de remplacement.

Avec le bouton Remplacer vous pouvez remplacer la notice sur laquelle vous vous trouvez par une autre.

  • Cliquez sur le bouton et rechercher le titre propre de la notice de remplacement.

  • Cliquez sur Remplacer.

Remplacer par Z39.50⚓

Vous pouvez également remplacer cette notice par une autre récupérée par Z39.50.

  • Cliquez sur Remplacer par Z39.50.

  • Cochez la ou les bibliothèques souhaitées pour la recherche. Si l'ISBN de votre notice était rempli, il s'inscrira automatiquement dans le champ Critères de recherche.

Remarque

Remplacer par Z39.50 suit la même ergonomie que Catalogue > Externe : Z39.50.

Pour plus d'informations, consultez la partie : Catalogue > Externe : Z39.50 dans ce guide.

Z39.50

Remplacer par une source externe⚓

Le bouton Remplacer par une source externe permet le remplacement d'une notice par une interrogation de connecteurs (Entrepôt OAI, Amazon, etc.).

Pour utiliser cette action vous devez avoir paramétré des connecteurs dans l'onglet Administration.

Pour plus d'informations, consultez la partie : Recherche > Externe : Connecteurs

Connecteurs

Le bouton Récolteur permet la mise à jour de la notice sélectionnée à partir de différente sources externes.

Dupliquer⚓

Vous pouvez dupliquer une notice pour cataloguer plus rapidement les différentes notices d'une série (BD, ...).

  • Cliquez sur Dupliquer.

Un nouveau formulaire apparaît avec tous les champs pré-remplis. Il reste simplement à modifier les champs différents pour le nouveau document.

Créer notice fille⚓

  • Cliquez sur Créer une notice fille

Un nouveau formulaire de notice apparaît. Dans la nouvelle notice affichée, seul le lien avec la notice mère est pré-rempli.

  • Complétez la notice pré-remplie et Enregistrer

Pré-visualiser la notice en OPAC⚓

Vous pouvez voir comment la notice apparaît dans votre OPAC en cliquant sur l'icône 

Impression d'un résultat de recherche⚓

Vous pouvez imprimer, éditer dans un document de traitement de texte ou envoyer par mail les résultats d'une recherche en cliquant sur l'icône de l'imprimante

Étendu de l'impression :

Les notices de la page courante : génère uniquement les notices de la page sur laquelle vous étiez avant de cliquer sur l'imprimante

Toutes les notices du résultat de recherche: génère toutes pages de notices issues de la recherche

Format des notices

Si des templates de notices sont paramétrés vous pouvez choisir celui qui sera utilisé comme format d'affichage des notices

Format court: n'affiche pas le résumé de la notice

Format long: affiche le résumé de la notice lorsqu'il existe

Imprimer l'entête de a notice: imprime la première ligne en gras de la notice qui reprend le titre et l'auteur

Afficher le permalink: affiche l'url de la notice dans l'opac

Imprimer la vignette /couverture: affiche l'image de la notice si cette dernière apparaît en gestion

Informations exemplaires

Imprimer les exemplaires : affiche les références des données d'exemplaires sous la notice

Documents numériques

Imprimer les documents numériques: affiche les références des documents numériques liés à la notice

Ne pas imprimer les documents numériques: n'affiche pas les données des documents numériques

Ne pas imprimer les exemplaires: n'affiche pas les données des exemplaires

Format de sortie

Imprimante: permet d'imprimer directement les résultats de recherche avec la mise en page définie

Traitement de texte: génère les résultats de recherche dans le logiciel de traitement de texte installé sur votre ordinateur

Envoyer par mail: permet d'envoyer un résultat de recherche à une adresse mail renseignée dans la case en dessous

Les informations complémentaires sous la notice⚓

  • les exemplaires

  • les documents numériques 

  • la possibilité d'ajouter une réservation

  • les réservations en cours sur le document (si elles ont été activées)

  • l'ajout d'exemplaires ou de document numériques
