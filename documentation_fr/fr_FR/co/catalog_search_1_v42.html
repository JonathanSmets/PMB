Catégories / Indexation décimale

Sommaire
  1. Naviguer parmi les catégories
  2. Chercher une catégorie
  3. Autopostage
  4. Chercher une indexation décimale

Naviguer parmi les catégories⚓

Les catégories de premier niveau sont affichées en bas de l'écran (elles sont modifiables dans l'onglet autorités).

Cliquez sur l'icône en forme de dossier en début de ligne pour afficher les sous-catégories se trouvant sous la catégorie sélectionnée.

Lorsque vous avez atteint la catégorie ou sous-catégorie désirée cliquez sur son libellé pour afficher les résultats.

Chercher une catégorie⚓

Vous pouvez également saisir dans le formulaire en haut de l'écran, sous Catégorie/index. décimale, votre recherche parmi les catégories. Dans les résultats, le nombre de notices utilisant cette catégorie est affiché en bout de ligne.

Autopostage⚓

L'autopostage permet de voir toutes les notices appartenant à une catégorie, ainsi que celles appartenant aux catégories parentes et enfants en fonction du paramétrage.

Si vous avez activé l'autopostage, voila des exemples de résultats de recherche : avec 1 niveau en dessous du terme "informatique", on obtient 4 résultats, avec 2 niveaux en dessous du terme informatique, on obtient un résultat supplémentaire.

Vous pouvez donc modifier à la main les niveaux de recherche dans les catégories.

Chercher une indexation décimale⚓

Sous le champ de recherche des catégories, vous pouvez effectuer une recherche portant sur l'indexation décimale des notices du catalogue.

En cliquant sur le bouton ... vous pouvez sélectionner dans un pop-up une indexation décimale afin d'afficher les notices correspondantes dans la base.
