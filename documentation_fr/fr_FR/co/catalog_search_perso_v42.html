Recherches prédéfinies

Présentation⚓

Les recherches prédéfinies sont des recherches multi-critères qui ont été construites puis enregistrées. Elles ont vocation à servir régulièrement.

Mode d'emploi⚓

Créer et enregistrer une recherche prédéfinie⚓

  • Catalogue > Recherche : Prédéfinie > Toutes les recherches

  • Cliquez sur Ajouter

  • Construisez votre recherche

  • Cliquez sur Mémoriser en recherche prédéfinie.

  • Complétez le formulaire de saisie

    Nom : identifie la recherche

    Raccourci du nom : définit un lien dans le sous-menu pour retrouver plus rapidement certaines des recherches prédéfinies

    Actif dans le bandeau de recherche : cochez pour voir la recherche dans le bandeau de recherche des recherches prédéfinies.

  • Cliquez sur Ajouter

Utiliser une recherche prédéfinie⚓

Catalogue > Recherche : Prédéfinie > Toutes les recherches

Cliquez sur la ligne de la recherche.

La recherche enregistrée apparaît, cliquez sur Rechercher pour avoir le résultat de cette recherche.

Modifier et supprimer une recherche prédéfinie⚓

Catalogue > Recherche : Prédéfinie > Toutes les recherches

Dans la ligne correspondant à la recherche à modifier ou à supprimer, cliquez sur Modifier.

Modifiez le nom, le raccourci du nom ou l'activation dans le bandeau de recherche et cliquez sur Enregistrer.

Pour supprimer la recherche, cliquez sur Supprimer.

Si vous cliquez sur la ligne de la recherche, PMB affichera la requête formulée sous forme de duplicata. De cette manière, vous pouvez compléter ou modifier une requête existante pour en créer une seconde.
