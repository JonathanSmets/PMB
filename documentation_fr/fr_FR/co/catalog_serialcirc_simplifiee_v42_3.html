Paramétrage d'une liste de circulation simplifiée

Il faut au préalable avoir créé un modèle prévisionnel et un abonnement. Pour savoir comment procéder, consulter les rubriques L'onglet modèles prévisionnels et L'onglet abonnements.

Au moment de paramétrer la liste de circulation, cocher la case : "Simplifiée" :

Ajouter ensuite des destinataires via le bouton Ajouter un destinataire puis Enregistrer.

Un destinataire doit être composé : d'un groupe intitulé : Direction - Service

D'un responsable : personne physique du module de circulation.

Remarque

Vous pouvez mettre autant de destinataires que souhaité. Vous n'avez pas besoin de paramétrer la fiche de circulation, cette dernière est générée différemment.

Pour imprimer les étiquettes de circulation, rendez vous dans le module d'éditions à la rubrique Imprimer des étiquettes de circulation simplifiées.
