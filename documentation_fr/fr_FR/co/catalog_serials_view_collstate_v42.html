L'onglet état des collections

Introduction⚓

Les collections de périodiques des bibliothèques sont souvent lacunaires : pertes plus fréquentes que pour les ouvrages, destructions, vols, pas d'abonnements certaines années, désabonnements.

ConceptEtat des collections⚓

L'état des collections permet notamment de mettre en avant :

  • les bornes des collections (collection de telle date à telle date, achevée ou encore en cours)
  • les lacunes
  • l'emplacement et les éventuelles conditions de consultation

Attacher une ou plusieurs collections à une notice de périodique permet de synthétiser ces informations pour les lecteurs sous la notice du périodique.

Une collection (c'est-à-dire une ligne dans l'état des collections) de périodique est définie au moins par :

  • un emplacement

  • un contenu de collection

  • un support

  • un statut

Les autres éléments sont facultatifs.

ConceptEmplacement de collection⚓

Cet emplacement permet de définir où est localisée, physiquement, la collection.

ConceptSupport de collection⚓

Le support définit la forme physique de stockage de la collection de périodique.

ConceptStatut de collection⚓

Le statut définit la visibilité de la collection affichée sous la notice dans l'OPAC.

Remarque

L'utilisation de l'état des collections nécessite un paramétrage : création des supports, statuts, ...

Pour plus d'informations, voir le Guide de l'administrateur PMB.

Etats des collections

Mode d'emploi⚓

L'ajout d'une collection à un périodique se fait depuis l'onglet Etat des collections présent sous la notice mère, à côté des onglets Bulletins, Abonnements et Modèles prévisionnels.

ProcédureComment ajouter un état des collections à une revue ?⚓

  1. Affichez la notice de périodique

  2. Cliquez sur l'onglet Etat des collections

  3. Cliquez sur Ajouter une collection

  4. Complétez le formulaire d'ajout d'état des collections

    • Localisation (selon paramétrage)

      Si les collections sont localisées, vous pouvez attribuer une localisation à la collection (même liste de localisations que pour les exemplaires).

    • Emplacement

      Cet emplacement permet de définir physiquement où est localisée la collection dans une liste déroulante.

    • Collection

      Le champ collection permet de saisir l'étendue de la collection concernée.

      Le bouton Calculer parcourt les bulletins déjà saisis pour le périodique et détecte les exemplaires non saisis, il sert à vous aider à établir l'étendue de la collection. Attention, il n'est utile que lorsque l'intégralité de la collection a été bulletinée dans PMB.

      Les bulletins non exemplarisés sont considérés comme des trous dans la collection.

      La présentation est faite sous la forme d'intervalles : N°début (date) – N°fin (date)

      Les différentes séquences sont présentées séparées par des points virgules.

      Par exemple, on peut imaginer :

      N°288 (juin 2004) - N° 297 (mai 2005) ; N°300 (septembre 2005) - N° 335 (septembre 2008)

    • Cote

      La cote que portent les numéros de périodiques de la collection, ou celle qui figure sur les boîtes d'archives, etc.

    • Archive

      Permet d'indiquer si une partie de la collection indiquée est archivée. Par exemple : 1 – 170 dans une collection 1-353.

      Ceci permet de présenter en une seule ligne la collection d'un périodique même si elle est fragmentée. C'est une alternative au fait de séparer la collection archivée de la collection accessible en créant deux états des collections.

    • Origine

      Origine de la collection : permet de signaler des dons ; de signaler le mode d'acquisition du périodique : achat au numéro ou par abonnement gratuit ou payant ; de signaler le dépôt d'une collection en provenance d'un autre organisme (dépôt d'une association tierce dans un centre de documentation, d'une BDP dans une bibliothèque publique, ...), etc.

    • Note

      Toute information complémentaire.

    • Lacunes

      Permet de mentionner des lacunes dans les séquences de numéros inscrits dans la collection. Il s'agit ici de présenter dans le champ Collection les plages pour lesquelles le périodique a été acquis. Ceci permet de mentionner explicitement les bulletins manquants (à noter : les exemplaires dont le statut est perdu, manquant inventaire, etc. sont intégrés dans le calcul de l'état des collections) ; ce champ vous permet de les repérer.

    • Support

      Le support permet d'indiquer une forme physique de conservation de la collection. Ce support est distinct du support paramétrable pour les exemplaires. On peut s'en servir pour indiquer que le support de l'exemplaire est inclus dans un support de stockage : mise des exemplaires de périodiques dans des boîtes d'archives ; reliure par année...

    • Statut

      Le statut de l'état des collections est comparable à celui de la notice: il en résulte la visibilité dans l'OPAC.

  5. Enregistrer

Pour modifier un état de collection déjà enregistré, cliquez sur le libellé de l'état de collection correspondant.

ProcédureModifiez un état de collection⚓

  1. Affichez la notice de périodique

  2. Cliquez sur l'onglet Etat des collections

  3. Cliquez sur la ligne de la collection à modifier

  4. Modifiez les informations dans le formulaire et Enregistrer ou Supprimer la collection

L'état des collections d'une revue est affiché dans le site sous la notice du périodique (visibilité selon le statut affecté).

ExempleExemple de mise en œuvre : deux collections pour un périodique⚓

Examinons ensemble le cas du stockage de numéros de la revue Géo dans un centre de documentation.

La revue Géo est conservée dans le centre de documentation uniquement pour deux ans. Les numéros précédents sont stockés dans une réserve.

Pour la première collection (qui est conservée dans le centre de documentation)  :

  • Emplacement : présentoir

  • Collection : 418 (décembre 2013)-425 (juillet 2014)

  • Cote : PER GEO

  • Origine : abonnement en cours

  • Note : dernier numéro exclu du prêt

  • Support : revue

  • Statut : empruntable

Pour la collection en réserve :

  • Emplacement : Réserve

  • Collection : 350 (avril 2008)- 417 (novembre 2013)

  • Cote : GEO

  • Origine : abonnement

  • Lacunes : 370 - 382 (suspension d'abonnement) ; 390 (non rendu par le lecteur)

  • Support : revue

  • Statut : consultation sur place
