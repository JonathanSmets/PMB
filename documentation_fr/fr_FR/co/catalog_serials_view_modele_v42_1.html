Ajouter un modèle

ProcédureComment ajouter un modèle ?âš“

  1. Depuis la notice du périodique, cliquez sur Modèle prévisionnel puis Ajouter un modèle

  2. Définir les paramètres généraux du modèle

    Un modèle se définit par :

    • un nom

    • une périodicité de base, choisie parmi les périodicités définies dans l'onglet Administration

    • une durée d'abonnement

    • une date de début et une date de fin du modèle

    Conseil

    Pour la durée du modèle, ne pas hésiter à mettre une durée longue, par exemple 10 ans.

    Ceci permettra de créer rapidement un nouvel abonnement l'année suivante, en utilisant le modèle existant.

  3. Définir si besoin des exclusions de date

    L'exclusion de dates permet de définir des moments (certaines semaines dans le mois, certains mois dans l'année par exemple) où il n'y aura pas de réception. Cochez ces dates en cas de besoin.

    Exemple

    Utilisez l'exclusion de dates pour :

    • un mensuel qui n'est pas publié en août

    • un quotidien qui ne paraît pas le dimanche

  4. Personnalisez si besoin la numérotation

    La numérotation est basée sur trois types de numérotations : numéro, volume et tome.

    Au bout d'un certain nombre de numéros (ou d'une certaine période), le numéro volume sera incrémenté (volume + 1), et sur le même principe pour un nombre de volumes, le numéro de tome sera incrémenté (tome + 1).

    Attention

    Le terme employé par le périodique pour sa numérotation (tome, volume, année, etc.) peut différer de la numérotation employée dans PMB.

    Ainsi, un numéro appelé "tome" sur la couverture du bulletin pourra être calculé à l'aide de la numérotation volume.

    Format d'affichageâš“

    Il est possible de définir dans le format d'affichage un format de numéro et un format de date. Ces deux valeurs rempliront les champs numéro et libellé de période du bulletin.

    Les variables $VOL; et $NUM; seront remplacées respectivement par le numéro de volume et le numéro, lors de l'enregistrement du bulletin. La variable $DATE ; sera remplacée par la date du jour au format AAAA-MM-JJ.

  5. Cliquez sur Enregistrer

    Après avoir saisi les paramètres généraux, défini ou non des exclusions de dates, défini la numérotation et les formats d'affichages, cliquer sur Enregistrer.

  6. Un calendrier apparaît sous le modèle, il est vide.

  7. Cliquer sur Générer la grille

    Cliquer sur Générer la grille pour définir le calendrier prévisionnel de réception du périodique.

Remarque

Vous n'avez pas besoin de cliquer sur Enregistrer après avoir généré la grille, le bouton Enregistrer sert à prendre en compte les modifications dans le formulaire situé au dessus. Toute modification dans le calendrier est immédiate : clic sur Générer la grille, modification en cliquant sur une date dans le calendrier, ...

ExempleExemple de numérotationâš“

Le périodique "Cursus" est trimestriel. Le numéro daté "Automne 2002" a une numérotation de type "Tome 5, numéro 3".

  • La périodicité de base est trimestrielle, le jour de début du modèle sera le 1er avril 2007, le modèle sera calculé sur 10 ans.

  • Pour le "numéro" , on utilisera la numérotation "numéro" de PMB, elle est cyclique et varie de 1 à 4. Elle revient à 1 tous les ans pour le numéro daté "Printemps""

  • Pour le "volume", on utilisera la numérotation "volume" (un cran juste au dessus de numéro). Elle augmente de 1 tous les ans pour le numéro daté "Printemps".
