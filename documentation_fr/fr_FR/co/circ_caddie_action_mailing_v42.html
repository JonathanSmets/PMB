Mailing

Le Mailing permet d'envoyer un mail à l'ensemble des lecteurs contenus dans un panier.

Remarque

Le mail peut être écrit dans un format HTML, qui permet une mise en forme (présentation).

Dans ce cas, l'utilisation d'un éditeur HTML est possible.

Pour plus d'informations, voir le Guide de l'administrateur PMB.

Paramétrer l'éditeur javascript HTML

  • Cliquer sur le panier contenant la liste des lecteurs destinataires.

  • Rédiger votre mail.

Détail des champsâš“

  • Renseigner l'Objet du mail et le Corps du message.

  • Pièce jointe : Vous pouvez ajouter une pièce jointe à votre mail. Cliquer sur le bouton Parcourir et sélectionner le fichier à joindre à l'envoi du mail.

  • Information de l'emprunteur : PMB vous permet de personnaliser l'envoi de vos messages en y incluant des informations emprunteur (nom, prénom, mot de passe, mail, prêts en cours,...).

    • Placer le curseur dans le corps du mail, à l'endroit où vous souhaitez que l'information soit incrémentée.

    • Sélectionner dans la menu déroulant l'information emprunteur que vous souhaitez renseigner dans le mail.

    • Cliquer sur Insérer.

    • Un codage s'ajoutera dans votre corps de mail qui permettra d'afficher l'information demandée dans le corps du mail (ex. : !!empr_first_name !! afin d'afficher le prénom du lecteur).

Template de mailâš“

Vous pouvez inclure un template de mail (modèle de mail) dans le corps de mail. Pour cela, il est nécessaire que celui-ci soit préalablement créé dans l'onglet Administration > Modules : Template de mail.

Le template vous permet de réutiliser un modèle de présentation de mail similaire lors de vos différents envois (logo, en-tête,...).

Pour en savoir plus, consulter le chapitre Administration.

Template de mail.

Sélectionnez dans le menu déroulant le template que vous souhaitez voir appliquer à votre mail.

Cliquez sur Insérer le template.

Le modèle sera désormais inséré dans le corps du mail.

Attention

Le menu déroulant template, ne s'affiche que si un template a été préalablement créé dans l'onglet Administration.

  • Cliquer sur Envoyer.

RemarqueRemise à zéro des envoisâš“

Lors du mailing, les lecteurs pour lesquels l'envoi s'est correctement déroulé sont pointés.

Le lien Remise à zéro des envois permet d'effacer ce pointage.
