Gestion des paniers

Circulation > Paniers : Gestion > Gestion des paniers

Ce sous-menu présente la liste des paniers auxquels l'utilisateur a accès. Il permet de créer, modifier et supprimer un panier.

Chaque ligne correspond à un panier où sont indiqués :

  • son nom (avec éventuellement un commentaire),

  • le nombre total d'éléments qu'il contient et le nombre d'éléments pointés parmi ceux-ci.

  • le menu déroulant Actions, disponible depuis la version 4.2, est en réalité un raccourci vers les actions disponibles sur les paniers

  • le menu déroulant symbolisé par le bouton permet d'appliquer un classement au panier

    Les paniers sont regroupés par classement.

Pour modifier ou supprimer un panier, cliquez sur Modifier.

Mode d'emploi⚓

Créer un panier⚓

Pour ajouter un nouveau panier, cliquez sur Créer un panier.

  • Nom du panier : identifie le panier
  • Commentaire : indication supplémentaire sur le panier
  • Autorisations  : l'utilisateur admin est coché par défaut ainsi que l'utilisateur qui crée le panier.
  • Classement du panier: permet de sélectionner un classement pour le panier en affichage dans la liste des paniers.
  • Accès rapide: permet de définir si le panier doit apparaître en accès rapide sur une page de liste de résultats ou non.

Cliquez sur Enregistrer.

Attention

N'oubliez pas de cocher les autorisations pour les utilisateurs qui ont le droit de voir et de modifier ce panier.

Voir le contenu d'un panier⚓

Pour voir le contenu d'un panier cliquez sur son nom dans la liste des paniers.

Si vous cliquez sur le nom d'un lecteur, vous accédez directement à sa fiche.

Pour retirer un élément du panier, cliquez sur l'icône  .

Les éléments inconnus et pointés⚓

  • éléments inconnus : ce sont des documents qui ont été collectés et qui ne sont pas reconnus par PMB

    Ceci permet de garder une trace de cette collecte.

  • éléments pointés : pour pointer un élément du panier, cliquez sur : . Une fois le document pointé, le compteur s'incrémente et l'icône devient :

    Ces éléments seront vus en détail dans la partie Pointage.
