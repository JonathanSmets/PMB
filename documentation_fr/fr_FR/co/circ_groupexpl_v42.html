Prêt d'un lot de documents

Depuis la version 4.0 PMB permet de prêter des exemplaires en un seul lot. La gestion des groupes se gère en Circulation >Groupe d'exemplaires.

Des groupes d'exemplaires peuvent être constitués pour le prêt de malle thématique ou de sac surprise.

Pour utiliser cette fonctionnalité, il est nécessaire d'activer un paramètre dans l'onglet Administration.

Groupe d'exemplaire

Une fois le paramétrage effectué un nouvel item sera visible dans le menu vertical de l'onglet Circulation.

Constituer le groupe d'exemplaires⚓

Créer un groupe d'exemplaires⚓

Cliquer sur Ajouter pour créer un nouveau groupe d'exemplaires.

Renseigner les informations nécessaires à la création du groupe d'exemplaires.

Nom : Nommer le groupe d'exemplaires

Statut de l'exemplaire principal : Sélectionner un statut qui sera automatiquement attribué à l'exemplaire qui servira à effectuer le prêt.

Le statut sélectionné doit permettre le prêt de document.

Statut des autres exemplaires : Sélectionner un statut qui sera attribué automatiquement aux exemplaires constituant le groupe.

Le statut sélectionné doit empêcher le prêt de documents.

Commentaire : Ce champ permet d'indiquer des informations complémentaires sur le groupe d'exemplaires. Uniquement visible en gestion.

Cliquer sur le bouton Enregistrer.

Conseil

Afin d'identifier plus simplement les exemplaires appartenant à des groupes, il est conseillé de créer deux nouveaux statuts d'exemplaires :

  • un statut servant à identifier l'exemplaire principal

  • un statut servant à identifier les éléments constituant le groupe

Ajouter des exemplaires à un groupe⚓

  • Accéder en mode modification au groupe d'exemplaires en cliquant sur son nom

Pour chaque ajout d'exemplaires au groupe :

  • Saisir le code-barres de l'exemplaire dans le champ Ajouter un exemplaire au groupe

  • Cliquer sur Suivant

Le document est bien associé au groupe lorsque :

  • la présence du message Cet exemplaire appartient maintenant à ce groupe sous le champ de saisie de l'exemplaire s'affiche

  • l'exemplaire est présent dans le tableau

Déterminer l'exemplaire principal⚓

Une fois le groupe d'exemplaires constitué, il est nécessaire de déterminer l'exemplaire qui permettra d'effectuer le prêt et retour du groupe. Il s'agit de l'Exemplaire principal.

Cocher le bouton radio correspondant à l'exemplaire désigné Exemplaire principal.

Cliquer sur Enregistrer .

Truc & astuce

Deux cas de figures pour définir l'élément principal du groupe :

  • S'il s'agit d'une série (BD, suite,...), il est conseillé de désigner le premier tome de la série en tant qu'Exemplaire principal.

  • S'il s'agit d'une malle/ d'un sac thématique, créer un exemplaire pour le contenant (malle ou sac) et le désigner en tant qu'Exemplaire principal.

Supprimer un exemplaire d'un groupe⚓

Il est possible d'extraire un document d'un groupe (demandé individuellement par un lecteur, perdu,...) en :

  • accédant au mode édition du groupe d'exemplaires

  • cliquant sur la croix située sur la colonne de droite,

L'exemplaire ne sera plus associé au groupe.

Truc & astuce

Lorsque le groupe d'exemplaires est constitué, il est conseillé d'ajouter à chacun des membres du groupe un message d'exemplaire pour signifier l'appartenance de l'exemplaire à ce groupe. Ce message apparaîtra lors des prêts et des retours d'exemplaires et apportera des éléments lors du prêt ou du retour d'un exemplaire du groupe.

Rechercher et filtrer dans les groupes d'exemplaires⚓

Recherche d'un exemplaire⚓

PMB permet d'effectuer une recherche sur un exemplaire afin de retrouver son groupe d'appartenance, d'apporter des modification à ce groupe, de pointer les éléments du groupe.

  • Scanner ou renseigner le numéro d'exemplaire du document

  • Cliquer sur Suivant

Filtrer les groupes⚓

Il est également possible de retrouver facilement des groupes en utilisant les filtres proposés :

  • sur les groupes en cours de prêts

  • sur les groupes incomplets

  • sur la localisation des groupes

Prêt d'un groupe de document⚓

ConseilPointer le groupe d'exemplaires avant le prêt (facultatif)⚓

Avant d'effectuer le prêt du groupe d'exemplaires, vous pouvez vérifier la présence de l'ensemble des documents dans le groupe en effectuant un pointage.

Le prêt d'un groupe d'exemplaires s'effectue de la même manière que lors d'un prêt classique.

Le prêt doit être effectué à partir de l'élément principal du groupe.

  • Se placer sur la fiche du lecteur à qui vous souhaitez attribuer le prêt.

  • Saisir le code-barres de l'exemplaire Élément principal.

Lorsque le prêt de document est effectué :

  1. l'ensemble des documents du groupe apparait dans un tableau. L'exemplaire principal est affiché en rouge.

  2. l'élément principal du groupe apparaît dans la tableau des prêts en cours.

Retour d'un groupe de document⚓

Truc & astuce

Avant d'effectuer le retour du groupe d'exemplaires, il est fortement conseillé de vérifier la présence de l'ensemble des documents du groupe en effectuant un pointage. Vous pourrez ainsi accepter le retour du groupe uniquement si le groupe est rendu au complet.

Le retour d'un groupe d'exemplaire s'effectue de la même manière que lors du retour d'un seul document. Le retour doit être effectué à partir de l'élément principal du groupe.

  • Se placer dans l'onglet Retour de documents

  • Saisir le code-barres de l'exemplaire Élément principal du groupe.

    Lors du retour d'un groupe d'exemplaires un tableau affiche l'ensemble des éléments du groupe et un message apparaît en rouge Rendu !.

Cas particulier : ⚓

Si le retour du document est effectué sur un des exemplaires du groupe qui n'est pas l'élément principal, PMB vous affichera un message d'alerte bloquant le retour du document.

Pointage des exemplaires⚓

Le pointage des exemplaires d'un groupe vous permet de valider la présence de l’élément au sein du groupe (avant le prêt et/ou au retour du groupe).

Accédez au module Groupes d'exemplaires, puis cliquez sur le bouton Voir présent sur la ligne du groupe que vous souhaitez pointer.

Pour pointer :

  • Saisir le numéro d'exemplaire dans le champ Code-barre de l'exemplaire

  • Cliquer sur Suivant (sauf si vous utilisez une douchette)

L'exemplaire est pointé.

Vous pouvez vérifier le pointage de l'exemplaire grâce à la présence d'une croix sur la colonne Pointé de l'exemplaire.

Éditer : pour accéder au mode édition du groupe d'exemplaires

Annuler : pour quitter le groupe d'exemplaire

Dépointage des exemplaires⚓

Avant d'effectuer un nouveau pointage, il est conseillé de dépointer les exemplaires du groupe.

PMB permet de dépointer :

  • un à un les exemplaires de votre groupe

  • l'ensemble des exemplaires du groupe

Dépointer un à un les exemplaires d'un groupe⚓

Il suffit de cliquer sur la croix située à droite du tableau en face de chaque exemplaire.

Dépointer l'ensemble des exemplaires d'un groupe⚓

Cliquer sur le bouton Tout dépointer.
