Créer et modifier un cadre du portail dans une page

Créer un nouveau cadreâš“

Remarque

Avant toute création de cadre, assurez-vous d'être sur la page qui l'accueillera dans la zone de visualisation.

Procédure

Pour créer un cadre du portail, il faut être dans le menu Portail > Construction : Construire.

  1. Cliquez sur le volet Modules.

  2. Cliquez sur un des modules pour accéder à une fenêtre popup de paramétrage du cadre et de son contenu.

  3. Définissez les conditions d'affichage.

    • Nom du cadre : obligatoire, il vous sert, par la suite, à le repérer parmi la liste des autres cadres de la page.

    • Cadre fixe ? : le cadre est présent sur toutes les pages Portail comme celles de l'OPAC (cadre recherche, cadre agenda...). Ce champ n'est à cocher que s'il n'y a pas de condition d'affichage.

    • Attaché à la page ? : permet, dans la partie construire, d'afficher la page attachée en même temps que la fenêtre de modification du module.

    • Mode de mise en cache : le cache permet de garder temporairement en mémoire le cadre à son premier affichage par un navigateur. Cela permet de gagner du temps de chargement lorsque la page sera à nouveau appelée par un navigateur.

      La durée de ce cache est paramétrable, pour en savoir plus consultez le chapitre Activer et configurer le Portail dans PMB du guide de l'administrateur.

      Un cadre ne doit pas avoir de condition d'affichage pour pouvoir utiliser un mode de mise en cache systématique.

    • Classe CSS : permet d'ajouter une classe sur le cadre

    • Conditions d'affichage :

      • Présence d'une variable globale : les variables sont les informations passées dans les URL des pages (un type de page, un type de recherche, un menu du compte emprunteur...),

      • Personnes authentifiées : la page est réservée aux personnes connectées,

      • Choisissez une Page : page du portail qui ont déjà été créées,

      • Type de page de l'OPAC : page de l'OPAC PMB,

      • Valeur d'une variable globale : valeur d'une variable placée en paramètre de l'URL (identifiant de la page, identifiant de la rubrique...),

      • Vues OPAC : limite le module à une ou plusieurs vues OPAC.

  4. Indiquez la source de données du cadre.

    La source de données dépend du module choisit. En choisissant une source de données, des champs supplémentaires apparaissent.

    Dans cet exemple la source de données est Rubriques :

    • Choisissez un sélecteur pour la source de données : il est possible de choisir directement dans le cadre les rubriques du contenu éditorial qui serviront à alimenter ce cadre. Pour une utilisation générique du cadre, une autre solution consiste à utiliser une variable d'environnement.

      • Sélectionnez une variable d'environnement : la variable d'environnement utilise les variables prédéfinies de la page qui recevra le cadre.

    Dans certains cadres vous avez la possibilité de filtrer le contenu que vous récupérez grâce à la source de données.

  5. Paramétrez l'affichage du contenu.

    L'affichage de vos éléments découlera de la vue que vous choisissez. Dans une majorité de cas, l'affichage dépendra du template obtenu à l'aide d'une combinaison des balises h2o mélangées à des balises HTML.

    Remarque

    Pour en savoir plus sur la construction d'un template, consultez le chapitre sur les vues des modules de ce même guide.

  6. Pour créer le cadre et pouvoir le positionner, cliquez ensuite sur Enregistrer. Une fois enregistré, il faudra alors le placer dans la page puis Enregistrer.

Modifier ou supprimer un cadreâš“

Procédure

Pour modifier un cadre du portail, il faut être dans le menu Portail > Construction : Construire.

  1. Affichez dans la zone de droite la page qui contient votre cadre

    Vous avez deux méthodes pour accéder à la page qui contient votre cadre :

    • dans la zone de droite, déplacez vous dans le portail jusqu'à atteindre la page qui vous intéresse (pour les pages du portail ou OPAC),

    • dans la zone de gauche, cliquez sur le volet Pages et cliquez sur le nom de la page qui contient votre cadre (uniquement pour les pages du portail).

  2. Cliquez sur le volet Objets définis si vous n'y étiez pas déjà.

  3. Modifiez le cadre en cliquant sur l'icône de crayon visible devant le nom du cadre.

  4. Effectuez les modifications que vous souhaitez et enregistrez votre cadre ou, supprimez le cadre en cliquant sur le bouton Supprimer tout en bas du cadre.

    Attention

    PMB ne vous demande pas une confirmation de suppression lorsque vous appuyez sur Suppprimer.

Truc & astuce

Quand vous avez beaucoup de cadres sur une page, il peut être nécessaire de classer vos cadres en utilisant la liste déroulante accessible derrière le nom d'un cadre.

Dans le volet Objets définis du menu Portail > Construction : Construire, cliquez sur l'icône puis choisissez un classement déjà existant ou tapez votre nouveau classement. Cliquez sur Enregistrer pour valider le classement.
