Gérer la liste des versions

Le portail de PMB permet de gérer un système de versioning. Ainsi, l'utilisateur peut revenir, à tout moment, à une ancienne version de son portail.

Remarque

Cette fonctionnalité ne recrée pas un cadre présent dans une ancienne version qui aurait été supprimé entre temps.

Revenir à une ancienne version⚓

Procédure

Pour charger une ancienne version du portail, il faut être dans le menu Portail > Construction : Construire.

  1. Déployez le volet Liste des Versions.

  2. Cliquez sur le  devant le CMS désiré.

  3. Cliquez sur la Date de création de la version à laquelle vous souhaitez revenir.

    Remarque

    Il est possible de supprimer des versions en utilisant le bouton disponible à la fin de chaque version.

  4. Puis Enregistrer

Ajouter un CMS⚓

Procédure

Un CMS dans PMB est une vue du portail. Chaque CMS possède les mêmes cadres positionnés différemment.

Pour créer un nouveau CMS, il faut être dans le menu Portail > Construction : Construire.

  1. Dans la zone de visualisation du portail à droite, assurez-vous d'afficher la version que vous souhaitez enregistrer dans le nouveau CMS.

  2. Rendez-vous dans le volet Liste des Versions.

  3. Cliquez sur le bouton Ajouter un cms.

  4. Complétez le formulaire :

    • le nom du CMS est obligatoire (en bleu),

    • un commentaire de présentation du CMS peut être ajouté (en rouge).

  5. Cliquez sur Enregistrer.
