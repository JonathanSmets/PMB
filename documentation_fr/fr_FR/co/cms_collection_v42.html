Gestion des collections numériques

Une collection numérique permet de regrouper un ensemble de documents numériques.

ProcédureAjouter une collection⚓

  1. Cliquez sur Ajouter une collection.

  2. Donnez un titre à votre collection.

  3. Éventuellement, vous pouvez inscrire un commentaire afin de décrire son contenu.

  4. Définissez une méthode de stockage.

    Remarque

    Les lieux de stockages des documents numériques sont paramétrés dans le menu Administration > Documents numériques. Consulter Stockages

  5. Puis enregistrez.

ProcédureAjouter des documents numériques à la collection⚓

Remarque

Tous les types de documents numériques sont acceptés : images, textes, vidéos...

  1. Dans la liste des collections numériques, cliquez sur le bouton Consulter la liste en fin de ligne.

  2. Importez les fichiers dans la collection en utilisant la méthode du glisser-déposer.

  3. Vous avez terminé l'import lorsque vous obtenez 100%.

ProcédureRenommer un document numérique, compléter ses métadonnées⚓

  1. Dans la liste des collections numériques, cliquez sur le bouton Consulter la liste en fin de ligne.

  2. Cliquez sur le nom du document.

  3. Remplissez les informations proposées.

    • Titre,

    • Description : commentaire associé au document,

    • Vignette : permet de forcer une vignette pour le document,

    • Forcer l'icône de la vignette en : permet de donner une icône par défaut s'il n'est pas possible d'en calculer une, penser à cocher Recalculer la vignette pour que ça soit pris en compte,

    • Recalculer la vignette : cette fonctionnalité permet de recalculer la vignette si vous n'avez pas d'aperçu de l'image importée,

    • URL du fichier : permet d'y accéder, peut-être copié et mis ailleurs,

    • Association : PMB indique si le document est utilisé dans un objet du contenu éditorial (rubrique ou article). Dans la cas où le document n'est pas utilisé, cette partie est vide.

    Remarque

    Aucune information est obligatoire.

    Remarque

    Pour que les vignettes soient calculées, il faut que le logiciel ImageMagick® ait été installé sur le serveur qui héberge PMB.

  4. Enregistrez les modifications ou cliquez sur la croix en haut à droite de la fenêtre popup pour annuler les changements.

ProcédureSupprimer un document numérique d'une collection⚓

  1. Dans la liste des collections numériques, cliquez sur le bouton Consulter la liste en fin de ligne.

  2. Cliquez sur le nom du document.

  3. Cliquez sur Supprimer.

    Vous ne pouvez pas supprimer un document numérique utilisé dans des rubriques ou articles.

    Si le document numérique est associé à un objet du contenu éditorial (rubrique ou article), cochez les associations et cliquez sur Supprimer les associations cochées.

    Avant de supprimer le document numérique, vous pouvez vérifier dans quel contexte il est utilisé en cliquant sur le titre de la rubrique ou de l'article qui l'utilise.

ProcédureSupprimer une collection numérique et tout ses documents associés.⚓

  1. Cliquez sur le titre de votre collection numérique.

  2. Cliquez sur Supprimer.

    Remarque

    Vous ne pouvez pas supprimer une collection numérique si elle contient des documents numériques rattachés.
