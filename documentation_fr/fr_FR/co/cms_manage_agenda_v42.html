Module Agenda

Le module agenda permet d'afficher un calendrier ou une liste d'événements où apparaissent les différents événements à venir (manifestations culturelles, formations, etc.).

Général⚓

Dans ce sous-menu, vous pouvez créer des types d’événements visibles dans le ou les agendas de votre portail.

Pour le paramétrage d'un événement, cliquez sur Ajouter un calendrier et complétez les informations demandées :

  • Nom : nom de votre type d'événement (visible dans votre portail),

  • Couleur : nom d'une couleur en hexadécimal (visible dans votre agenda),

  • Type d'article à utiliser : type de contenu de vos articles qui servira à associer un article à son type d'événement,

    • Champ à utiliser pour le début d'un événement : champ personnalisé de votre type de contenu qui sera utilisé dans l'agenda comme date de début,

    • Champ à utiliser pour la fin d'un événement : champ personnalisé de votre type de contenu qui sera utilisé dans l'agenda comme date de fin.

Remarque

Pour en savoir plus sur les types de contenu, consultez le chapitre contenu éditorial de ce même guide.

Pour modifier un événement, cliquez sur son nom, modifiez le puis enregistrez.

Pour supprimer un événement, utilisez l'icône située derrière son nom.

ExempleExemple de paramétrage d'un module Agenda dans un cadre du portail⚓

Dans la partie construire, on ajoute un élément permettant d'afficher un agenda sur une page.

Il faut bien penser à sélectionner les modules créés précédemment pour qu'ils s'affichent dans l'agenda (cf zone en rouge).
