La rubrique avancée

La rubrique avancée, rubrique possédant le plus de champs, est celle qui permet de faire des descriptions complexes.

Elle permet, par exemple, de faire la description de chaque bibliothèque.

Identifiant de la localisation⚓

Cet identifiant, à chercher dans Administration > Administration : Exemplaires > Localisation, permet de faire un lien entre la page créée et le navigateur de sections.

Horaires d'ouvertures⚓

Ce champ permet de créer un bloc dédié dans la page, appelé Horaires d’ouverture.

Coordonnées⚓

Ce bloc permet, à l'instar des horaires d'ouverture, de créer un bloc spécifique avec un titre.

Lien(s)⚓

Ce champ permet de mettre des liens directs sur la page.

Au moment de l'ajout, les liens sont vérifiés pour ne pas mettre de liens morts sur la page.

Inscrire dans le champ Libellé le texte sur lequel doit être le lien.

Le + en bout de ligne permet de répéter le champ.

Multimédia⚓

Ce bloc interprète le code que l'on inscrit à l'intérieur.

Il permet, par exemple, de mettre une carte dynamique ou une vidéo.

Étagère associée (carrousel)⚓

Sélectionner dans ce champ une étagère qui sera affichée sous forme de carrousel en bas de la page.

Flux RSS⚓

En inscrivant dans ce champ l'adresse d'un flux, les dix derniers items de ce flux apparaîtront sur la page.

A la fin, ne pas oublier de cliquer sur le bouton Enregistrer
