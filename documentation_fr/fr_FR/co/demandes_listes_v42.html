Toutes les demandes⚓

Cette liste recense toutes les demandes enregistrées dans la base, quel qu'en soit le statut, même archivé, et quel que soit l'utilisateur auquel elles ont été attribuées.

Depuis cet écran, il est possible de :

  • filtrer les demandes affichées,

  • ajouter une nouvelle demande,

  • modifier une demande en cliquant sur la ligne correspondante,

  • supprimer une demande.

Demandes à valider⚓

Cette liste affiche toutes les demandes dont le statut correspond à à valider.

Depuis cet écran, il est possible de :

  • filtrer les demandes affichées,

  • ajouter une nouvelle demande,

  • modifier une demande en cliquant sur la ligne correspondante,

  • supprimer une demande,

  • passer une ou plusieurs demandes dans l'état Validé,

  • passer une ou plusieurs demandes dans l'état Refusé.

Remarque

Les demandes attribuées mais pas encore validées sont également affichées dans ce menu.

Demandes validées⚓

En plus des actions déjà disponibles pour les autres listes, il est possible de :

  • passer une ou plusieurs demandes dans l'état Validé,

  • passer une ou plusieurs demandes dans l'état Refusé.

Demandes refusées⚓

Ce menu contient la liste des demandes refusées et attribuées à l'utilisateur courant.

En plus des actions déjà disponibles pour les autres listes, il est possible de :

  • passer une ou plusieurs demandes dans l'état Fini,

  • passer une ou plusieurs demandes dans l'état Abandonné,

  • passer une ou plusieurs demandes dans l'état Validé.

Cette dernière action revient à récupérer une demande précédemment refusée.

Demandes finies⚓

En plus des actions déjà disponibles pour les autres listes, il est possible de :

  • archiver les demandes finies,
  • créer une question de la FAQ à partir d'une demande.

Demandes abandonnées⚓

En plus des actions déjà disponibles pour les autres listes, il est possible d'archiver les demandes abandonnées.

Demandes archivées⚓

Ce menu permet de consulter les demandes archivées.

En plus des actions déjà disponibles pour les autres listes, il est possible de valider à nouveau une demande.

Remarque

Il n'est pas nécessaire de valider à nouveau une demande pour pouvoir agir sur celle-ci : modifier les informations de la demande, créer des actions, etc.

L'état archivé permet essentiellement d'avoir un suivi de ces demandes et d'effectuer des statistiques.

Demandes non assignées⚓

En plus des actions déjà disponibles pour les autres listes, il est possible d'affecter plusieurs demandes à un même utilisateur.
