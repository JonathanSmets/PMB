Équations

Ce menu permet de visualiser les équations de recherche qui alimentent les bannettes lors des DSI.

A partir de ce menu, il est possible de modifier, supprimer une équation déjà enregistrée et d'en créer de nouvelles.

Pour chercher une équation, saisissez son nom dans la case de recherche située en haut de l'écran.

Afin de limiter votre recherche, vous pouvez utiliser le sélecteur de classement situé à droite.

ProcédureCréation d'une nouvelle équation⚓

  1. Effectuer la recherche multi-critères

    Depuis la liste des différentes équations de recherche, cliquez sur le bouton Ajouter.

    Vous êtes alors redirigé vers l'interface de recherche multi-critères.

  2. Enregistrer la formule comme "équation de recherche".

    Si le résultat de la recherche multi-critères vous convient, cliquez sur le bouton Transformer en équation DSI pour sauvegarder la formule de recherche.

    PMB vous redirige ensuite dans l'onglet DSI.

  3. Enregistrer l'équation de recherche

    Remplissez le formulaire pour sauvegarder l'équation en indiquant un nom (avec éventuellement un classement et commentaire, interne aux utilisateurs) :

    Remarque

    Ici, vous n'enregistrez pas les résultats de la recherche que vous venez d'effectuer, mais uniquement la formule qui vous permet d'arriver à ces résultats.

ProcédureModification d'une équation⚓

  1. Accéder aux propriétés de l'équation.

    Depuis la liste des différentes équations, cliquez sur celle que vous souhaitez modifier.

  2. Effectuer la modification.

    Pour modifier la formule de l'équation, cliquez sur le bouton Modifier cette recherche.

    PMB vous dirige alors vers la recherche multi-critères. Après avoir lancé la recherche, cliquez sur Sauvegarder équation DSI pour enregistrer la modification.

    Cliquez sur Enregistrer pour sauvegarder l'équation modifiée.
