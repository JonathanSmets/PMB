Code-barres : Génération libre

Cet outil permet d'éditer des pages de code-barres, imprimables sur des planches d'étiquettes.

Utilisation⚓

Le nom de la bibliothèque apparaît en fonction du nom que vous avez indiqué dans la localisation.

Vous pouvez :

  • Générer automatiquement

    Indiquez le n° du premier code-barres que vous souhaitez éditer ainsi que le nombre à générer.

  • Générer à partir d'un fichier

    Avec le bouton Parcourir sélectionnez votre fichier sur votre disque dur, votre clé usb, etc.

    Pour modifier la mise en page des étiquettes cliquez sur Afficher puis adaptez les paramètres à vos besoins.

    Vous pouvez vous servir d'un fichier texte contenant les numéros à imprimer, personnaliser le nom de votre centre documentaire,etc.

    Une fois les code-barres générés, la mise en page est conservée. Lorsque vous imprimerez à nouveau des code-barres, vous retrouverez la mise en page qui avait été configurée auparavant.

    Quand vous avez réglé tous ces paramètres, vous pouvez cliquer sur Générer.

Résultat⚓

Un document PDF est généré, vous pouvez l'imprimer.

Conseil

Choisissez des étiquettes de bonne qualité et imprimez les code-barres avec une imprimante laser pour en améliorer la longévité.
