L'onglet "Mes prêts / Mes réservation".

Depuis la version 4.2 de PMB, l'onglet Mes Prêts / mes réservations a été fusionné.

On peut donc, depuis cet onglet, avoir accès à :

Prêts en cours :

Afficher les prêts en cours

Affiche le tableau des prêts en cours regroupés par localisation.

Afficher les prêts précédents

Permet d'afficher les anciens prêts si la fonctionnalité a été activée en partie gestion de PMB.

Export Tableur

Apparaît après l'activation du paramètre empr_export_loans en partie gestion de PMB. Permet de générer un tableur contenant les prêts en cours

Afficher les réservations:

Affiche les réservation en cours. On retrouve ici les fonctionnalités de l'ancien onglet "Mes réservations" des versions précédentes de PMB
